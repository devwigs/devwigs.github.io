Motion in a Uniform Electric Field

You learned in Physics 20 that an object that experiences a constant force would move in a parabola-shaped trajectory. This type of motion is called projectile motion.

  • This is because the kinematics equation below is quadratic when \(\vec{\mathbf{a}}\) is constant:$$\Delta{\vec{\mathbf{d}}} = {\vec{\mathbf{v}}}_i \Delta t + {1\over 2}{\vec{\mathbf{a}}}\left(\Delta t\right)^2$$
  • An object with a constant charge moving through a uniform electric field must also follow a parabolic trajectory since the force (and therefore the acceleration) would be constant.
  • The constant acceleration is calculated using Newton’s 2nd Law:$$\vec{\mathbf a} = {{\vec{\mathbf F}}_e + {\vec{\mathbf F}}_g \over m} = {q{\vec{\mathbf E}} + m{\vec{\mathbf g}}\over m}$$
  • Typically, we would set up an experiment so that the electric and gravitational fields are both in the y-direction, so that \({\vec{\mathbf a}}_x = 0\).
  • Often (but not always) the electric force is so strong that the gravity term has no effect on the answer to the precision of our calculations.
