expand_lessImages

You can insert images such as PNG or JPEG image files into your HTML document using the img tag.

<body>
<p>Here is a picture of a rocket...</p>
<p><img src="https://learn.davidmaccarthy.repl.co/media/rocket.png"
  alt="Picture of a Rocket"/></p>
<body>

The src (source) attribute describes the URL of the image file and the alt attribute contains text to be displayed when the image is unavailable. The source URL can be either an absolute or a relative URL.

You can use the width and height attributes to specify the size of the image. This will help your browser leave space for the image while it is waiting for it to download.

<body>
<p>Here is a picture of a rocket...</p>
<p><img src="https://learn.davidmaccarthy.repl.co/media/rocket.png"
  alt="Picture of a Rocket" width="128" height="127"/></p>
<body>

expand_moreVideo

HTML5 introduced a video tag for embedding video files in web pages. Because different browsers may support different video formats, you can list multiple versions of the video in the source tags which are children of the video tag.

<video width="640" height="360" controls="1">
  <source src="https://learn.davidmaccarthy.repl.co/home/cs20_30/web1/img/movie.mp4" type="video/mp4">
  <source src="https://learn.davidmaccarthy.repl.co/home/cs20_30/web1/img/movie.ogv" type="video/ogg">
</video>

expand_moreYouTube

Rather than upload videos to your own web server, many web pages rely on video hosting services such as YouTube to provide embedded video. By including videos in this manner, you do not need to worry about the video format. To embed a YouTube video in your web page, visit the video’s YouTube page and obtain the HTML code from the Share/Embed section.

<iframe width="640" height="360" frameborder="0" allowfullscreen="1"
  src="https://www.youtube.com/embed/Vd5duzNtBuI"></iframe>
