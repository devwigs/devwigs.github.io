expand_lessfor Loop

We saw that Python’s while statement is used to run a block of instructions repeatedly as long as the Boolean condition we provide remains True. We also saw that flags and counters are useful techniques for stating the Boolean condition.

Another type of loop available in Python is a for loop. For-loops are used to iterate through a sequence. One type of sequence is a string; Python treats each string value as a sequence of individual characters.

# Save this program as 's14_ex1.py'

word = "Salisbury Composite High School"
for letter in word:
    print(letter)

Run the program above. You will see that it prints each of the letters on its own line. The loop works by assigning the variable name letter to the first value in the sequence, in this case the letter S. The indented block is then executed on this value. Once the block is complete, letter is assigned the next value in the sequence, a, and the indented block is repeated. This loop continues until each of the values in the sequence has had its turn as letter.

Python allows “nested” loops and conditionals. This means that you can have an if statement within a for loop, for example. The program below will print each of the lower case letters and punctutation marks on its own line, but will ignore any upper case letters.

# Save this program as 's14_ex2.py'

word = "Salisbury Composite High School"
for letter in word:
    if letter == letter.lower():
        print(letter)

Each level of nesting requires an additional level of indentation; the if statement is indented to show it is part of the for loop, and the print statement is indented again to show it is part of the if block.

Besides strings, other types of sequences that you can iterate through include files and lists of numbers or other data.

# Save this program as 's14_ex3.py'

data = 0, 1, 1, 2, 3, 5, 8
for num in data:
    print(num)

expand_morerange Function

Python has a built-in range function that can be used to produce sequences of integers. For example, calling range(10) produces the sequence of numbers from 0 to 9. This gives you an alternate way to loop with a counter. The following example runs the loop ten times, with the variable count keeping track of how many times the loop has run.

# Save this program as 's14_ex4.py'

for count in range(10):
    print(count)

This is more concise than using a while loop with a counter, because you do not need to initialize or increment the count variable.

You can pass more arguments to the range function to create other sequences of numbers. For example, if you want to count by twos, starting at 10 and ending at (but not including!) 20...

# Save this program as 's14_ex5.py'

for count in range(10, 20, 2):
    print(count)
