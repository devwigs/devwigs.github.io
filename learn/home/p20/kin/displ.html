An object’s position (\(\vec{\mathbf d}\)) means its location.

  • In Physics, positions can be described using one or more coordinates that represent a location relative to an arbitrary origin.
  • Motion along a straight line is called one-dimensional (1D) because it can be described with only one coordinate axis.

Displacement is defined as the change in an object’s position over some interval of time:$$\Delta{\vec{\mathbf d}} = {\vec{\mathbf d}}_f - {\vec{\mathbf d}}_i$$

  • Displacement is measured in a straight line, even if the object’s trajectory (path) is not straight.
  • The distance that an object travels is measured along the trajectory, and is not the same as its displacement.

A quantity that involves a direction as well as a magnitude is called a vector. A quantity that has a magnitude only (no direction) is called a scalar.

  • Displacement is a vector, as are force and velocity.
  • Distance is a scalar, as are time, mass, and energy.
