The physics module defines a Sketch subclass named Environment. This class simulates a physical environment in which (non-disabled) sprites that have a mass property with a positive value exhibit collision behaviour approximating the laws of physics. Here is an example...

from scropr.sketch import Sprite, WRAP, BOUNCE
from scropr.physics import Environment
from scropr.image import Image
from random import uniform

def setup(sk):

    # Load background and monster images
    sk.setBackground("bg.png")
    img = Image("costume.png")

    # Create 10 random monster sprites
    for i in range(10):
        velocity = uniform(-3, 3), uniform(-3, 3)
        posn = sk.randPixel()
        mass = uniform(0.5, 2)
        h = sk.height / 15 * mass
        Sprite(sk, img, height=h, mass=mass, radius=True, edge=(WRAP, BOUNCE),
            elasticity=0.98, posn=posn, velocity=velocity)

# Run the sketch
Environment(setup).run()

If you create a custom drawing function, be sure to call the physicsDraw method instead of simpleDraw to preserve the physics-based behaviour.
