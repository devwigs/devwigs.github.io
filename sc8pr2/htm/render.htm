sc8pr.Renderable Class

The Renderable class is an “abstract” class that implements an image property by calling a render method only when the current rendering is marked as stale. Subclasses must implement the render method for instances to function correctly. In addition to the attributes inherited from Graphic it defines the following...

Assignable Properties

stale: A bool indicating whether the graphic needs to be re-rendered.

Read-Only Properties

image: A pygame.Surface containing the rendered, scaled, and rotated graphic.

Instance Methods

refresh() → self: Checks if the current rendering is stale and, if so, re-renders immediately rather than waiting until the graphic needs to be drawn.
