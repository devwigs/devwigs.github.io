sc8pr.robot.gui.Robot Class

The sc8pr.robot.gui module defines a Robot class that adds additional methods to the sc8pr.robot.Robot class, to allow the robot to communicate with its user via synchronous dialogs.

In addition to the attributes it inherits from sc8pr.robot.Robot, the sc8pr.robot.gui.Robot class defines the following...

Instance Methods

confirm(prompt, title=None, response=True) → bool: Runs a message box and returns a bool indicating which button was used to dismiss the dialog; True for Okay and False for Cancel.

prompt: A message to display.

title: The message box title.

response: Specifies whether a response (Okay/Cancel) is required.

textinput(prompt, title=None, allowCancel=False, num=False) → str, float, or bool: Prompt the user for some input. Returns the user response as a str or float (depending on whether numerical input was required), or False if the dialog is cancelled by the user.

prompt: A message to display.

title: The message box title.

allowCancel: A bool indicating whether the user can cancel the dialog.

num: Specifies whether a numerical response is required.

numinput(prompt, title=None, allowCancel=False) → float or bool: Calls textinput with the num argument set to True.
