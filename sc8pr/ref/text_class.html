The Text class represents text that can be drawn into a canvas. In addition to the attributes it inherits from Graphic and Renderable, the Text class defines the following...

Constructor

Text(data)

data: The initial value of the text to be drawn. This can be a str instance or an object that can be converted to a string by the str function.

Assignable Properties

It is recommended that you call the config method to modify properties rather than using a direct assignment. This will ensure that the text is re-rendered when needed.

align: One of the constants LEFT, RIGHT, or CENTER (defined in the sc8pr module) specifying the alignment when the text contains more than a single line; default LEFT.

bg: Color of the text background. If None, the background will be transparent.

border: Color of the rectangular border to draw around the text; default is black.

color: Color to use for the text; default is black.

font: The name of a system font or path to a font file. Specifying None (default) uses the pygame default font.

fontSize: The size of the font in pixels; default 24.

fontStyle: One of the constants BOLD, ITALIC, or BOLD+ITALIC specifying the font style; the default is 0 indicating neither bold not italic. This attribute applies only to system fonts.

height: The rendered size of the text is determined by the font and the text content. Assigning a new value to the height or width will cause the font size to be adjusted, but the size of the final rendering will be only approximate to what was requested. Assigning a value to the size attribute will raise an exception.

padding: Number of extra pixels to add around the outside of the rendered text but inside the border; default 0.

spacing: Number of extra pixels to add between lines for multi-line text; default 0.

weight: Thickness in pixels of the text border; default 0.

width: The rendered size of the text is determined by the font and the text content. Assigning a new value to the height or width will cause the font size to be adjusted, but the size of the final rendering will be only approximate to what was requested.

Read-Only Properties

size: The rendered size of the text.

Instance Methods

render() → pygame.Surface: Renders the text. This method is called automatically by Renderable.image. To force re-rendering, set the Text instance’s stale attribute to True.
