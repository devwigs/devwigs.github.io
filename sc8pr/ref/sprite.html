Classes

Sprite (BaseSprite): Class representing animated bitmap images.

Functions

collide_rect_circ(left, right) → int: Checks whether the objects are colliding based on their blit rectangles first, and if so then re-checks based on circular shape. Returns 0 if false or 1 if true.

left, right: Sprites, or other objects with a rect and radius property.

collide_rect_mask(left, right) → int: Checks whether the objects are colliding based on their blit rectangles first, and if so then re-checks based on their masks. Returns 0 if false or 1 if true.

left, right: Sprites, or other objects with a rect and mask or image property.

elasticCircles(mass1, mass2) → True or None: Adjusts sprite motion properties based on an elastic collision between circles. Returns True if adjustments are made; i.e. if there is a collision.

mass1, mass2: Sprites with a mass property.

physics(sk, model=elasticCircles) → list: Updates motion properties for colliding sprites that have a mass property. Calls the sprites oncollide handlers by layer order. Returns a list of colliding sprites.

sk: A reference to the sketch.

model: A function that determines whether two masses are colliding and updates their properties if so.
