sc8pr is an educational programming package for Python 3.4+ and pygame 1.9. Inspired by Scratch and Processing, sc8pr’s aim is to make it easy for new and experienced Python programmers to create animations, games, and other graphics-based programs.

    sc8pr (version 2.0) features include:

    • a simple, event-driven approach to creating interactive animations
    • image manipulation
    • sprite management
    • physics-based collisions
    • video effects / transitions
    • a selection of GUI controls
    • robotics simulator [experimental]

    Version 2.0 is brand new! Reference and tutorial pages are under construction.
