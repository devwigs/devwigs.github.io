Overview

sc8pr is a Python 3 package for educational programming. Inspired by Scratch and Processing, sc8pr’s aim is to make it easy for new programmers (and experienced ones too!) to create interactive Python programs with graphics and animations.

Features include:

  • a simple framework for creating interactive animations
  • sprite management
  • physics-based collisions
  • robotics simulations
  • image manipulation
  • graphical user interface
  • graph plotting

To learn more, select one of the blue links from the navigation panel along the left side of the page.

Requirements

The sc8pr package requires:

  • Python 3.2+
  • Pygame 1.9.1+

Please ensure that Python and Pygame are both installed before setting up sc8pr.

Try sc8pr

The sc8pr package is pure Python 3 code, so you can use it without installing as a site package...

  • Download the source code from GitHub.
  • Open the sc8pr-master subfolder.
  • Extract the sc8pr package into your project folder or PYTHONPATH.

Install with pip

To install using pip, run the following command ...

pip install https://github.com/dmaccarthy/sc8pr/archive/master.zip
