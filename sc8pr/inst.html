Please ensure that you have Python 3.4 or higher installed before proceeding.

Open your command prompt as an administrator and run the commands below. The pip3 program should be located in the Scripts folder of your Python 3 installation.

pip3 install setuptools wheel
pip3 install https://github.com/dmaccarthy/sc8pr/archive/master.zip

If pygame is not already installed, the setup will attempt to install it. If a pygame build cannot be located for your platform and Python version, you will need to install it manually before using sc8pr. Please visit the pygame website if this happens.
