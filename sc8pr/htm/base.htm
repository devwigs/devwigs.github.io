sc8pr.BaseSprite Class

BaseSprite is a Graphic subclass that implements the ondraw event handler to control the motion of the sprite, and adds several properties describing the sprite’s kinematics. This class also provides a “pen” feature to trace the motion of the sprite within its canvas. BaseSprite is an “abstract” class; use one of its subclasses to instantiate sprites.

Assignable Properties

acc: A 2-tuple or list giving the acceleration; the amount in pixels by which to adjust the vel property each frame.

bounce: One of the constants HORIZONTAL, VERTICAL or BOTH, specifying whether to bounce the sprite from the edges of the canvas. These constants are defined in the sc8pr module.

bounceType: One of the constants CIRCLE or RECT, specifying the bounce model; default is CIRCLE. [Added in v2.1]

drag: The “drag coefficient”. A value of 0.01 reduces the velocity and spin by 1% each frame.

spin: The number of degrees by which to adjust the angle property each frame.

pen: A 2-tuple or list giving the pen colour and thickness to use when tracing the sprite’s motion on the background; e.g. ("red", 3). You can assign a value of None to disable tracing. The pen is only available when the background is an image, not a uniform colour.

vel: A 2-tuple or list giving the velocity; the amount in pixels by which to adjust the pos property each frame.

wrap: One of the constants HORIZONTAL, VERTICAL or BOTH, specifying whether to wrap the sprite when it completely leaves the canvas.

Instance Methods

circleBounce(self) → None: Performs calculations to bounce the sprite from the edge of its canvas; called by ondraw. [Deprecated]

kinematics(self) → None: Performs kinematics calculations; called by ondraw.

ondraw(self) → None: Update the sprite’s motion properties each frame and (if applicable) trace its motion with the pen. Calls the sprite’s onbounce or onwrap handler if necessary.

onwrap(self, update) → None: Default onwrap handler; calls the penReset method.

update: A code (HORIZONTAL, VERTICAL or BOTH) identifying which edge(s) the sprite has crossed.

penReset(self) → None: Lifts the pen to allow wrapping or resizing.

simpleWrap(self) → None: Performs calculations to wrap the sprite when it leaves the canvas; called by ondraw.

toward(self, pos, mag=None) → tuple: Returns a vector directed from the sprite’s current position to the position specified. [Added in v2.1]

pos: The target position as a 2-tuple or 2-list.

mag: The magnitude of the vector; if None, the sprite’s current speed will be used.
