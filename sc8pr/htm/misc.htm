sc8pr.misc Package

Modules

cursors: Defines or imports (from pygame) a number of cursors: arrow, broken_x, circle, cross, crosshair, diamond, hand, menu, move, sizer_x, sizer_xy, sizer_y, text, thickarrow, tri_left, tri_right.

effect: Defines a number of classes for applying effects and transitions to graphics objects.

latex: Rendering LaTeX equations as images using codecogs.com.

pil: Adds additional image features (such as screen captures) that can be accessed when the Pillow (PIL) package is installed.

plot: Defines the Locus and Plot graphics classes.

progress: Defines the ProgressBar class.

video: Defines the Video class for representing video clips as sequences of compressed images, and the ImageIO class for encoding and decoding MP4 and other media formats (requires ffmpeg, and the imageio package plus all of its dependencies.)
