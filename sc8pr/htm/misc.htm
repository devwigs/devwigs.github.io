sc8pr.misc Package

Modules

cursors: Defines or imports (from pygame) a number of cursors: arrow, broken_x, circle, cross, crosshair, diamond, hand, menu, move, sizer_x, sizer_xy, sizer_y, text, thickarrow, tri_left, tri_right. These cursors can be assigned as the sketch’s cursor property.

effect: Defines a number of classes for applying effects and transitions to graphics objects.

hsv: Use HSV color and draw HSV color wheels or color matrices.

media: Defines classes for encoding/decoding media using external packages such as Pillow and imageio. Added in v2.2.

plot: Defines the Locus and Plot graphics classes. Deprecated as of v2.2.

table: Defines a Canvas subclass that can be used for arranging graphics into tables or grids. Added in v2.2.

progress: Defines the ProgressBar class.

video: Defines the Video class for representing video clips as sequences of compressed images.

webcache: Sends requests for images and other web-based data asynchronously, and manages the response cache. Allows for rendering of LaTeX markup as images using codecogs.com.
