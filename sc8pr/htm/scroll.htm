sc8pr.gui.scroll Module

Experimental in v2.2.

This module defines the ScrollCanvas and ScrollSketch classes which display a viewport to a virtual canvas and provide scroll bars to move the viewport. The abstract base class BaseScroll defines features shared by both classes. ScrollCanvas is also a subclass of Canvas while ScrollSketch is a subclass of Sketch. Additionally, the following are defined...

Constructors

ScrollCanvas(size, scrollSize=None, bg=None)

ScrollSketch(size=(512, 288), scrollSize=None)

size: The actual size of the canvas or sketch.

scrollSize: The “virtual” or scrollable size of the canvas; if omitted the actual canvas size will be used.

bg: The canvas background color or image.

Read-Only Property

scrollPos: A 2-tuple describing the virtual canvas coordinates of the pixel at the top left of the viewport.

Assignable Properties

Both classes change the default value of resizeContent to False. ScrollSketch also changes the default value of fixedAspect to False

resizeScroll: A bool specifying whether to scale the scrollSize automatically when the canvas is resized; default False.

scrollSize: A 2-tuple containing the “virtual” or scrollable size of the canvas.

Instance Method

scrollSnapshot() → Image: Renders an image of the entire “virtual” canvas.
