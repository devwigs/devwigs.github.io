sc8pr.gui.radio.Radio Class

Radio is an Options subclass where selecting one of the check boxes automatically deselects the others. It does this by implementing the onaction event handler, and triggering onchange when the selected option changes.

Usage example:

from sc8pr import Sketch
from sc8pr.gui.radio import Radio

def setup(sk):
    labels = "Yes", "No", "Maybe"
    sk += Radio(labels).config(pos=sk.center).bind(onclick, onchange)

def onchange(cv, ev):
    option = cv.selected.layer
    label = cv[option + len(cv) // 2]
    print("Change:", option, label.data)

def onclick(cv, ev):
    try:    # Click text
        print("Click:", ev.target.data)
    except: # Click background
        print("Click:", cv)

Sketch().play()

In addition to the attributes inherited from from Canvas and Graphic the Radio class defines the following...

Constructor

Radio(text, height=None, space=4, imgs=None, txtConfig={}): All arguments are the same as for Options.

Assignable Property

selected: A reference to the check box (Button instance) that is currently selected, or None if no option is selected.
