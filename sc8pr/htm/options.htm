sc8pr.gui.radio.Options Class

Options is a Canvas subclass for creating an options menu comprising check buttons and text. Check boxes are added to the canvas first, followed by the text labels. For example, if an Options instance has three items, the check boxes will have layer values 0 through 2 and the labels 3 through 5.

The Options class only defines a constructor; all other attributes are inherited from Canvas. You can bind an onaction event handler to detect clicks on enabled check boxes.

Usage example:

from sc8pr import Sketch
from sc8pr.gui.radio import Options

def setup(sk):
    labels = "Yes", "No", "Maybe"
    sk += Options(labels).config(pos=sk.center).bind(onclick, onaction)

def onaction(cv, ev):
    btn = ev.target
    option = btn.layer
    label = cv[option + len(cv) // 2]
    print(option, btn.selected, label.data)

def onclick(cv, ev):
    try:    # Click text
        print(ev.target.data)
    except: # Click background
        print(cv)

Sketch().play()

Constructor

Options(text, height=None, space=4, imgs=None, **kwargs)

text: A sequence of strings describing the option labels.

height: The height of the check boxes in pixels; if omitted, the check box size will be determined by the height of the rendered text labels.

space: Number of additional pixels to add between rows and around the outside of the canvas.

imgs: A sequence of images to use for the checkbox depending on its status ("normal", "hover", "selected", "hoverselected", "disabled"). If omitted, the default images will be used.

kwargs: Additional properties for rendering the text labels (font, fontSize, etc.).
