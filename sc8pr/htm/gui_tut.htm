GUI Controls

The sc8pr.gui subpackage defines several types of graphics objects that can be used to interact with the user. The example below shows how to use TextInput controls to obtain text data from the user asynchronously.

TextInput instances already have event handlers defined for onclick, onkeydown, and onblur events. These event handlers allow the text input to behave as intended. These handlers trigger the onchange event whenever the data in the text input changes, and the onaction event whenever the control loses focus, such as when a mouse click occurs outside of the text input. Your program can bind handlers to these events to access the data...

from sc8pr import Sketch
from sc8pr.gui.textinput import TextInput

# Event handlers
def onchange(gr, ev): print("Change:", gr.data)
def onaction(gr, ev): print("Action:", gr.data)

def setup(sk):
    # Create the TextInput
    name = TextInput("", "Enter Your Name")

    # Configure the TextInput
    name.config(pos=sk.center, fontSize=40, color="blue", promptColor="red")

    # Bind event handlers and add to the sketch
    sk += name.bind(onchange, onaction)
	
# Create and run the sketch
Sketch((384, 144)).play()

As with other types of graphics, you do not need to use the event names (onchange, onaction) as the names of the event handler functions. If you choose to use a different name, pass the handler to the bind method as a keyword argument rather than a positional one. For example, if you want to name the onaction handler submitName...

name.bind(onchange, onaction=submitName)
