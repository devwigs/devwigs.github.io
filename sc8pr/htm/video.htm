sc8pr.misc.video Module

Added in v2.1. The video module in v2.0 was experimental and does not conform to this API!

Classes

Video (Sprite): A class for storing and retrieving video recordings and other sequences of compressed images.

Grabber: A class for performing screen captures using the Pillow package. As of v2.2 this class has been moved to the sc8pr.misc.media module.

ImageIO: [Experimental] A wrapper to the imageio and imageio-ffmpeg packages for encoding and decoding video using ffmpeg. As of v2.2 this class has been moved to the sc8pr.misc.media module.
