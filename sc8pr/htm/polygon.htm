sc8pr.shape.Polygon Class

The Polygon class is used to draw and perform calculations related to polygons. In addition to attributes inherited from Shape and Graphic it defines the following...

Constructor

Polygon(pts, anchor=None)

pts: A sequence of points representing the vertices of the polygon.

anchor: Polygons do not use an anchor property in the way that most other graphics classes do; the polygon’s anchor property is used as the centre for translation, rotation and scaling. If no anchor argument is provided, the constructor will use the centre of the polygon’s containing rectangle. Alternatively, you can provide a 2-tuple or 2-list to specify a specific point, or an int for the index of a vertex to use as the centre.

Assignable Properties

anchor: Returns the polygon’s centre of rotation. When assigning new values, use pos or xy to translate the polygon, and anchor to change the centre of rotation without translating.

vertices: Redefine the polygon with new vertices. Use the setPoints method instead if you also want to change the anchor.

Read-Only Properties

center: Coordinates of the center of the polygon’s containing rectangle.

image: A pygame.Surface containing the current rendering of the shape.

segments: A list of Line instances representing the sides of the polygon.

size: The size of the polygon as the width and height of a containing rectangle.

Instance Methods

containsPoint(pos) → bool: Determines whether the given point, pos, is located within the polygon. This method does not account for the offset of the polygon’s canvas in the sketch; use the contains method for this.

intersect(other) → list: Calculates the points of intersection between the polygons self and other.

setPoints(pts, anchor=None) → self: Replace the existing vertices and and anchor properties by new values; see the constructor for details.

transform(self, rotate=0, scale=1) → self: Scales and rotates the polygon about its centre (pos). You can also perform rotations by assigning a new angle property, and scale transformations by modifying the width or height properties.
