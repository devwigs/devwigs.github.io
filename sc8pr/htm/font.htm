sc8pr.text.Font Class

Font is a “static” class for loading and caching font files.

Class Properties

cacheSize: The maximum number of fonts to keep in the cache. Once the cache is full, a font will be removed from the cache each time a new font is loaded; default 32.

log: When True (default), a message will be printed to sys.stderr when the font cache is full.

Static & Class Methods

dumpCache() → None: Empties the font cache.

find(*args) → str or None: Searches for a system font from among the args. returns the name of the first system font found, or None if none of the fonts are available.

get(name, size=24, style=0) → pygame.Font: Load the requested font or retrieve it from the cache.

name: The font name.

size: The size of the font in pixels.

style: One of the constants BOLD, ITALIC, or BOLD+ITALIC specifying the font style; the default is 0 indicating neither bold not italic. This attribute applies only to system fonts.

installed() → list: Returns a sorted list of system font names.

mono() → str or None: Searches the system fonts for one of several common monospace fonts.

sans() → str or None: Searches the system fonts for one of several common sans-serif fonts.

serif() → str or None: Searches the system fonts for one of several common serif fonts.
