sc8pr.misc.latex.LatexCache Class

Added in v2.0.1.

The LatexCache sends requests to the codescogs.com server in order to render LaTeX markup as images. Responses are saved as PNG files in a local folder.

Constructor

LatexCache(folder="./")

folder: The location in which to save the images as PNG files.

Read-Only Properties

index: A dictionary containing cached LaTeX markup as keys with an alias as the corresponding value. The alias is used to determine the names of the cached image files.

indexFile: The name of the JSON file to which the cache index is saved.

Instance Methods

cached(alias, dpi=128) → bool: Determines whether the specified alias exists in the cache at the specified resolution.

alias: A random sequence of characters assigned when the LaTeX markup is first rendered.

dpi: The resolution of the image in “dots per inch”.

file(name) → str: Uses os.path.abspath to return the absolute path of a file relative to the cache folder.

get(*args, dpi=128, update=True, onload=None): Renders and caches the LaTeX markup described by the args. Returns an image or a list of images, depending on the number of positional arguments supplied. The format of the images is determined by the onload argument. Usage example:

lc = LatexCache("./latex")
latex = "A = \\pi r^2", "C = 2 \\pi r"
area, circum = lc.get(*latex, dpi=256, onload=Image)

args: A sequence of strings containg the LaTeX to be rendered. It is faster to get many images at once rather than calling get multiple times since the get function sends all requests asynchronously.

dpi: The “dots per inch” argument determines the image size. You can set this to None if you prefer to set the dpi within your LaTeX code.

update: Specifies whether to save the modified index when rendering LaTeX not currently in the index. If calling get multiple times, you can improve the efficiency of your program by setting this to False and then calling the update method after all new renderings are complete.

onload: A callback function specifying how to process the pygame.Surface instances that are returned. Set this to sc8pr.Image to use the images in your sc8pr application.

png(alias, dpi=128) → str: Returns the PNG file name for the specified alias and image resolution.

update() → None: Saves the cache index as a JSON file.
