Installation

Please ensure that you have Python 3.4 or higher installed before proceeding. Next, open your terminal program / command prompt as an administrator. Finally, run the commands below.

Windows...

py -3 -m pip install -U pip setuptools sc8pr

Linux / Mac OS...

sudo pip3 install -U pip setuptools sc8pr

If you have multiple versions of Python 3 installed, you can specify the version you want to use by changing py -3 to py -3.6 (Windows) or pip3 to pip3.6, for example.

If pygame is not already installed, the setup will attempt to install it. If a pygame build cannot be located for your platform and Python version, you will need to install it manually before using sc8pr. Please visit the pygame website if this happens.
