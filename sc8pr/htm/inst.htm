Installation

Please ensure that you have Python 3.4 or higher installed before proceeding.

Next, open your terminal program / command prompt as an administrator. Finally, run the command below. For the most common Python configurations, this should upgrade Python’s pip and setuptools software to the latest version, and then install sc8pr 2.1 and pygame 1.9 from the Python Package Index.

Windows...

py -3 -m pip install --upgrade pip setuptools sc8pr==2.1a0

Linux / Mac OS...

sudo pip3 install --upgrade pip setuptools sc8pr==2.1a0

If pygame is not already installed and the setup cannot locate a build for your platform and Python version, you will need to install it manually before using sc8pr. Please visit the pygame website if this happens.
