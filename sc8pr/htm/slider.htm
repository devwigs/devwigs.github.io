sc8pr.gui.slider.Slider Class

Added in v2.1. The Slider class in v2.0 was experimental and does not conform to this API!

The Slider class is used to create slider GUI controls. This class implements event handlers for onkeydown and either onmousedown (v2.2) or onclick (v2.1), and triggers onchange. When onchange is called, the event argument will have a method property which specifies how the slider value was changed. This property will equal one of the constants CLICK, SCROLL, DRAG, or KEY, defined in sc8pr.gui.slider.

In addition to the attributes it inherits from Canvas, the Slider class defines the following...

Constructor

Slider(size=(128,16), knob=None, lower=0, upper=1, steps=0)

size: The size of the slider canvas as a 2-tuple or 2-list.

knob: A color or Graphic instance to use as the slider’s knob; defaults to "grey" in v2.1. In v2.2, this can be a 2-tuple or 2-list where the second color is used when the mouse is hovering over the knob. If omitted the knob will be created using the colors specified by the 2-tuple Knob.colors.

lower: The slider value when the knob is in the lowest position.

upper: The slider value when the knob is in the highest position.

steps: The number of increments in which to move the slider; if 0, the arrow keys will adjust the slider value by 0.01 of the total range, and the mouse will allow the slider knob to be positioned to the nearest pixel.

Read-Only Properties

methodNames: A list of strings describing the constants. [Added in v2.2]

knob: The graphic being used as the slider’s knob.

lower: The lowest allowed value of the slider.

upper: The highest allowed value of the slider.

Assignable Properties

allowButton: A tuple of mouse button numbers that will be considered clicks by the onclick event handler. Clicks by other mouse buttons will be ignored but will still be considered to have been handled. Default is (1, 4, 5).

val: The current value of the slider.

steps: The number of increments as described for the constructor.
