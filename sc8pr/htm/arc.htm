sc8pr.shape.Arc Class

The Arc class draws an elliptical or circular arc. In addition to attributes inherited from Ellipse this class defines the following...

Constructor

Arc(size)

size: The size of the ellipse as a 2-list or 2-tuple; or a single number for a circular arc.

Assignable Property

arc: A 2-list or 2-tuple specifying the starting and ending angles in degrees. The arc is drawn clockwise

Read-Only Property

fill: None; the Arc class does not support the fill attribute and an exception will be raised if a value is assigned.
