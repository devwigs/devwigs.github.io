sc8pr.misc.video.Grabber Class

Added in v2.1. The Grabber class in v2.0 was experimental and does not conform to this API!

This class performs screen captures on Windows or MacOS using the Pillow package. The example below captures a 640 × 360 rectangle starting at coordinates (100, 100). The screen grab is then converted to a sc8pr.Image instance.

g = Grabber([100, 100, 640, 360])
img = PixelData(g.grab).img

Constructor

Grabber(rect=None)

rect: The region of the screen to capture. Can be None to capture the entire screen, a 2-tuple or 2-list specifying the size of the rectangle to capture from the upper left, or a 4-list or 4-tuple giving both the upper left coordinates and the size.

Read-only Property

grab: The current contents of the specified rectangle as a Pillow image. Use PixelData to convert the capture to an image that can be used in sc8pr.
