sc8pr.misc.video.Grabber Class

Added in v2.1. The Grabber class in v2.0 was experimental and does not conform to this API!

This class performs screen captures using the Pillow package.

Constructor

Grabber(rect)

rect: A rectangle describing the region of the screen to capture.

Read-only Property

grab: The current contents of the specified rectangle as a PIL image. Use Image.fromPIL to convert the capture to an image that can be used in sc8pr.
