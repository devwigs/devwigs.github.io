sc8pr.gui.textinput.TextInput Class

The TextInput class is used to obtain text input from the user asynchronously. This class implements event handlers for onclick, onkeydown, and onblur. These events may trigger onchange or onaction. In the development version (v2.2.dev), ondraw refers to the scroll method.

In addition to the attributes it inherits from Text, the TextInput class defines the following...

Constructor

TextInput(data="", prompt=None)

data: A str describing the text input content.

prompt: A str to be displayed when the text input data is empty.

Assignable Properties

In sc8pr v2.0, the padding property inherits a default value of 0. This may result in the cursor becoming invisible when positioned at the end of the text input. For v2.1, the default padding is 4.

allowButton: A tuple of mouse button numbers that will be considered clicks by the onclick event handler. Clicks by other mouse buttons will be ignored but will still be considered to have been handled. Default is (1,).

cursorOn: The time in seconds that the cursor remains visible before blinking off; default is 0.35.

cursorTime: The total time in seconds of the cursor on-off blink cycle; default is 1.0.

data, prompt: As described for the constructor arguments.

promptColor: The text color of the prompt.

Instance Method

scroll(pix=None, rel=True) → self: Adjusts the position of the text input within its canvas. Called automatically by ondraw and onblur. [Experimenal in v2.2.dev]

pix: The number of pixels to scroll the text, positive for right and negative for left. If not specified it will be calculated automatically, provided the angle property is 0 or 90.

rel: Specifies whether to scroll the graphic relative to its current position (True) or its unscrolled position (False).
