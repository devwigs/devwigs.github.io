sc8pr.misc.plot.Locus Class

Added in v2.0.1. The Locus class in v2.0.0 was experimental and does not conform to this API!

The Locus class represents graphics described by an equation or a sequence of points. Loci are drawn directly onto the canvas. The example below produces the following output:

from math import pi, sin
from sc8pr import Sketch
from sc8pr.misc.plot import Locus

def setup(sk):
    lrbt = -10, 10, -1, 1
    param = -2*pi, 2*pi, 500
    wave = Locus(sin, lrbt, param)
    sk += wave.config(weight=2, stroke="red")

Sketch().play()

In addition to the attributes inherited from from Shape and Graphic the Locus class defines the following...

Constructor

Locus(data, lrbt, param)

data: A list or tuple of points, or a function from which points can be generated.

lrbt: A list or tuple specifying the coordinates corresponding to the left, right, bottom, and top edges of the plot area. If top is not specified, it will be calculated so that the y-axis has the same scale as the x-axis. If top and bottom are both omitted, the axes will have the same scales and the x-axis will be positioned in the middle of the plot.

param: A 3-list or 3-tuple specifying the starting and ending values of the parameter, and the number of steps. This is used to generate the data points when a function is provided as data. The param argument is ignored if data is a sequence of points. In the example above, the sin function is graphed from −2π to +2π by drawing 500 segments.

Assignable Properties

data, lrbt, param: See descriptions for the constructor arguments.

vars: A dictionary to be passed to the function that generates the data.

Read-Only Properties

pointList: A list of data points.

snapshot: This attribute is set to None to disable the snapshot method. To convert a locus into an Image, add the locus to a Canvas instance and then take a snapshot of the canvas.

Instance Methods

pointGen() → generator: A generator for the sequence of data points.
