sc8pr.misc.latex Module

Added in v2.0.1. The latex module in v2.0.0 was experimental and does not conform to this API!

This module renders LaTeX markup as images using codecogs.com. This example renders the equations for the area and circumference of a circle as a list of two Image instances:

eqns = "A = \\pi r^2", "C = 2 \\pi r"
imgs = render(*eqns, dpi=256, onload=Image)

Functions

render(*args, **kwargs): Returns a single image or a list of images, depending on the number of positional arguments supplied. The format of the images is determined by the keyword arguments.

*args: A string or sequence of strings containing the LaTeX markup to be rendered.

The following optional keyword arguments are supported:

dpi: The “dots per inch” argument (default 128) determines the image size.

raw: The codecogs server returns the image as PNG-formatted bytes. By default, this data will be converted to a pygame.Surface instance. Set the raw argument to True to prevent this conversion.

onload: This is a callback to perform additional processing on each image. Setting this to Image (as in the example above) will convert the rendering to an Image instance. You can provide a custom callback that takes a bytes or pygame.Surface instance (depending on the value of raw) as its argument.
