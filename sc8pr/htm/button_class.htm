sc8pr.gui.button.Button Class

The Button class allows you to use buttons as GUI controls. This class implements event handlers for onclick, onmouseover, and onmouseout. The onclick method may trigger onaction. In addition to the attributes it inherits from Canvas, the Button class defines the following...

Constructor

Button(size=None, options=None)

size: The size of the button; if None, the size of the options[0] image will be used.

options: A sequence or dictionary of images or colors to be used for the button backgrounds. Dictionary keys correspond to the button states: "normal", "hover", "selected", "hoverselected", "disabled". If omitted, the default colors (sc8pr.gui.button.OPTIONS) will be used. If an int is provided, a subset of the default colors will be used; for example, passing 2 will use the default normal and hover colors only.

Assignable Properties

allowButton: A tuple of mouse button numbers that will be considered clicks by the onclick event handler. Clicks by other mouse buttons will be ignored but will still be considered to have been handled. Default is (1,).

enabled: A bool indicating whether the button is enabled.

options: A list of colors or images describing the backgrounds for the different button statuses.

selected: A bool indicating whether the button has a selected status.

status: An int value between 0 and 4 indicating the current status of the button (normal, hover, selected, hoverselected, disabled).

Read-Only Properties

selectable: Checks that the button has a selected option and that it is not currently disabled.

statusName: A string describing the button status: "normal", "hover", "selected", "hoverselected", or "disabled".

Instance Method

textIcon(text, icon=None, padding=6) → self: Adds text and an optional icon to the button. For example, to create an “Okay” button: okay = Button(size, 2).textIcon("Okay", True)...

text: A string containing the button text or a Graphic instance to be used as text. The text will be added to the button after the icon and therefore can be referenced using a layer (index) of -1.

icon: A Graphic instance (such as an Image or Sprite) to be used as the icon. If a bool is passed, the default icon for “Okay/Yes” (True) or “Cancel/No” (False) will be used.

padding: The number of pixels of space to be added between the image and the edge of the button.

Static Methods

checkbox(imgs=None) → Button: Create a check box button.

imgs: A sequence of images or image file names to use as backgrounds; if None the default check box images will be used.

radio() → Button: Create a radio check box button. This method simply calls checkbox using the default radio images.
