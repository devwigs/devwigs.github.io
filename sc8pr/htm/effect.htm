sc8pr.misc.effect Module

Demo Source Code

Classes

Effect: Abstract base class for transitions and other effects.

ReplaceColor (Effect): Replace one color with another.

Tint (Effect): Decrease any of the RGBA values of the image.

Wipe (Effect): Apply a straight-line “wipe” transition from any side, corner, or center.

Squash (Wipe): Apply a squash transition from any side, corner, or center.

MathEffect (Effect): Apply a potentially non-linear “wipe” transition described by a mathematical relation.

WipeSlope (MathEffect): Apply a straight-line “wipe” transition from any corner with an arbitrary slope.

PaintDrops (MathEffect): Apply a paint drop transition.

Dissolve (Effect): Replace random pixels with transparency, a specific color, or random colors.
