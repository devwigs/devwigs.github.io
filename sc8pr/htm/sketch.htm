sc8pr.Sketch Class

The Sketch class represents the “top-level” canvas that contains all other sc8pr graphics objects. In addition to the attributes it inherits from Graphic and Canvas, Sketch defines the following...

Constructor

Sketch(size=(512,288))

size: The initial size of the sketch.

Assignable Properties

caption: The caption of the sketch window. [Added in v2.1]

capture: A sc8pr.misc.video.Video instance used to record the animation; default None. The video must have an interval attribute specifying how often to record frames.

cursor: A reference to the current cursor data. You can load cursor data from sc8pr.misc.cursors or create your own using pygame.cursors.

dirtyRegions: A list of pygame.Rect regions to be updated when drawing the next frame; you can set this to None to always update the entire screen. To resume tracking dirty regions after disabling, set this attribute to [sk.rect]; i.e. a list containing the sketch’s rect attribute as its sole item.

fixedAspect: When set to True (default); the sketch will resize itself to match the aspect ratio of its background image and will maintain the aspect ratio when being resized.

frameCount: The number of animation frames drawn; incremented at the beginning on each drawing cycle.

frameRate: The approximate number of animation frames to draw each second; default 60.

quit: A bool that can be set to True to terminate the animation.

realTime: A bool specifying whether to attempt to correct for actual frame rates slower than requested by the frameRate property; default False. [Experimental]

Read-Only Properties

anchor, pos: These attributes shoud be considered read-only and will become formally read-only in v2.1.1.

evMgr: An object that manages the sketch’s event loop. This object has the following read-only properties:

drag: A reference to the Graphic instance being dragged, or None.

focus: A reference to the currently-focused Graphic instance.

hover: A reference to the Graphic instance over which the mouse is currently hovering; if the mouse leaves the sketch window, hover will refer to the sketch itself.

Instance Methods

cover() → Image: Creates a semi-transparent image that can cover the sketch. This is useful for blocking mouse events, for example when running a dialog on top of the sketch.

play(caption="sc8pr", icon=None, mode=True) → self: Run the sketch.

caption: Text to use as the window caption.

icon: Path to an image file to be used as the window icon.

mode: A bool specifying whether the sketch window can be resized by the user.
