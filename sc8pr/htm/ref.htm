sc8pr Reference

The core of sc8pr is defined in the sc8pr, sc8pr.geom and sc8pr.util modules. The sc8pr module defines essential classes such as Sketch, Image, and Canvas. The sc8pr.geom module defines various 2D geometry-related functions, while sc8pr.util defines several helper and general purpose functions.

Additional modules in the main package define additional graphic classes. The sc8pr.shape module defines the Line, Circle, Polygon, and Arrow classes. The Text and Robot classes are defined in the sc8pr.text and sc8pr.robot modules respectively.

The sc8pr.gui subpackage defines additional graphics classes for creating Graphical User Interface controls. The sc8pr.misc subpackage defines additional features such as video effects and the Video, Plot, and Locus graphics classes.
