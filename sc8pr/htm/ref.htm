sc8pr Reference

This reference covers sc8pr versions 2.0 through 2.2. Please note that version 2.2 is currently an “alpha” release and is subject to change prior to the official 2.2.0 release.

Essentials

  • The sc8pr module defines essential classes such as Sketch, Image, and Canvas.
  • The sc8pr.geom module defines various 2D geometry-related functions.
  • The sc8pr.util module defines several helper and general purpose functions.

Common

  • The sc8pr.sprite module defines the Sprite class for creating interactive, animated graphics.
  • The sc8pr.shape module defines the Line, Circle, Polygon, and Arrow classes and their corresponding sprites.
  • The sc8pr.text module defines the Text and Font classes for adding text to your sketch.
  • The sc8pr.robot module defines features for creating robotics simulations. In v2.2 sc8pr.robot becomes a package containing several additional modules.
  • The sc8pr.gui package defines additional graphics classes for creating Graphical User Interface controls.

Extras

  • The sc8pr.misc package defines additional features such as video effects and the Video and Locus graphics classes. This package may also include experimental features to be included in future versions of sc8pr.
  • The sc8pr.examples package is deprecated as of v2.2. The robotics simulations have been moved to the sc8pr.robot package and the remaining examples have been moved online and are no longer included them with sc8pr itself.
