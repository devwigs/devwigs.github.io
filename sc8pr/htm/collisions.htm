sc8pr.sprite.Collisions Class

The Collisions class is used to detect collisions between BaseSprite instances.

Constructor

Collisions(cv, collide=collide_rect_circ)

cv: A reference to the canvas in which the collisions are occurring. [Renamed in v2.1 from sk in v2.0.]

collide: A function that determines whether or not two sprites are colliding.

Instance Methods

among(group=None) → dict: Test for collisions within a group of sprites. Returns a dictionary with colliding sprites as keys and the dictionary returned by the involving method as values.

group: The sprites to check. If None, all BaseSprite instances currently in the sketch are included. Sprites without a rect attribute (e.g. sprites that have been added but not yet drawn) are omitted for this and for the methods listed below.

between(group1, group2=None) → [list, list]: Test whether there are current collisions between two groups of sprites. Returns lists of sprites from each group as a 2-list.

group1: The first sequence of sprites to check for collisions.

group2: The second sequence of sprites. If None, all BaseSprite instances currently in the sketch are included.

betweenMap(group1, group2=None) → dict: Test whether there are current collisions between two groups of sprites. Returns a dictionary with pairs of colliding sprites as keys and the collide function return value (usually 1) as the value.

group1: The first sequence of sprites to check for collisions.

group2: The second sequence of sprites. If None, all BaseSprite instances currently in the sketch are included.

involving(sprite, group=None, asBool=False) → dict or bool: Test whether a particular sprite is currently colliding.

sprite: The sprite to be checked.

group: A sequence of sprites to check for collisions with sprite. If None, all BaseSprite instances currently in the sketch are included.

asBool: If True the method will return a bool indicating whether or not a collision is occurring. Otherwise, returns a dictionary with the colliding sprite instances as keys and the collide function return value (usually 1) as the value.
