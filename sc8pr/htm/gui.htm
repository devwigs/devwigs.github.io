sc8pr.gui Package

Added in v2.1. The gui package in v2.0 was experimental and does not conform to this API!

Modules in sc8pr.gui define a number of graphics classes for creating Graphical User Interface (GUI) controls inside your sketches. You can use the Canvas class to assemble controls into dialogs.

  • Button
  • MessageBox
  • Menu
  • Options
  • Radio
  • Slider
  • TextInput

You may also want to use tkinter dialogs, for example to choose a file or folder. To avoid conflicts bewtween pygame and tkinter, using the ask function from the sc8pr.gui.dialog module...

ask(dialog, allowQuit=True, **kwargs) → str or other value returned by dialog.

dialog: A dialog function such as tkinter.filedialog.askopenfilename.

allowQuit: The pygame.QUIT event will be disabled while the dialog is running. Set this argument to a bool to indicate whether or not to enable the event when the dialog is finnished. If you want to call ask while pygame is not initialized, set this to None.

kwargs: Additional arguments to pass to the function referenced by dialog.
