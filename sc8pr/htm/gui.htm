sc8pr.gui Package

Modules in sc8pr.gui define a number of graphics classes for creating Graphical User Interface (GUI) controls inside your sketches. You can use the Canvas class to assemble controls into dialogs.

  • Button
  • Menu
  • Options
  • Radio
  • Slider
  • TextInput

There is also a tkdialog module that defines features for using sc8pr with tkinter dialogs.
