sc8pr.geom Module

This module contains functions for performing 2D geometry calculations.

Functions

angleDifference(a2, a1=0) → float: Calculate the difference a2-a1 and express the result as an angle between −180 and +180.

a2, a1: Angles in degrees.

delta(v2, v1=None, mag=None) → tuple: Calculate the difference between two vectors, v2-v1, and optionally adjust the magnitude of the result.

v2, v1: The vectors to subtract as 2-tuples or 2-lists; v1 can be omitted, allowing the function to adjust the magnitude of v2.

mag: If specified, the desired magntiude of the result; for example, passing 1 will produce a unit vector.

dist(p1, p2=(0,0)) → float: Calculate the distance between two points.

p1, p2: Points as 2-tuples or 2-lists.

positiveAngle(a) → float: Express an angle as a value between 0 and 360.

a: Angle in degrees.

polar2d(vx, vy, deg=True) → tuple: Convert a vector to polar form, returning the magnitude and direction as a 2-tuple.

vx, vy: The components of the vector.

deg: A bool indicating the angle units: True for degrees and False for radians.

sprod(v1, v2) → float: Calculate the scalar product of two vectors.

v1, v2: Vectors as 2-tuples or 2-lists.

transform2d(pt, **kwargs) → tuple: Perform a transformation (shift, scale, rotate) on a single point.

kwargs: Arguments describing the transformation, to be passed to the transform2dGen function (see below).

transform2dGen(pts, mx=None, shift=(0,0), preShift=None, **kwargs) → generator: Perform a transformation (shift, scale, rotate) on a sequence of points. The kwargs arguments are used only when mx is None.

pts: The sequence of points to transform; as 2-tuples or 2-lists.

mx: A 4-tuple or 4-list representing a 2×2 matrix (linear transformation). If None, the function will construct the matrix from the kwargs data.

shift: A translation to be applied after the matrix, as a 2-tuple or list.

preShift: A translation to be applied before the matrix, as a 2-tuple or list. If this argument is True the preShift will be the negative of the shift, permitting rotations and scale transformations using a centre other than the origin.

kwargs["rotate"]: A rotation angle in degrees, when mx is None.

kwargs["scale"]: A scale factor as an int or float, or a 2-tuple or list if the x and y scale factors differ. Used only when mx is None.

kwargs["rev"]: The default (False) performs the scale transformation and then the rotation. Changing this to True rotates first and then scales. The order only matters when the x and y scale factors differ. Used only when mx is None.

vec2d(r, a, deg=True) → tuple: Create a 2D vector from polar form.

r: The magnitude of the vector.

a: The direction of the vector.

deg: A bool indicating the angle units: True for degrees and False for radians.

Constant

DEG: The float value π / 180.
