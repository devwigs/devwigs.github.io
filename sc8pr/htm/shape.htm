sc8pr.shape Module

Classes

Shape (Graphic): Abstract base class for geometric shapes. It defines the following assignable attributes for use by subclasses...

fill: The color to fill the shape with; default None.

stroke: The color to outline the shape with; default is black.

weight: The thickness in pixels of the shape outline; default 1.

Line (Shape): Represents line segments.

Circle (Shape): Represents circles.

Ellipse (Shape): Represents ellipses.

Arc (Ellipse): Represents elliptical arcs.

Polygon (Shape): Represents polygons.

Arrow (Polygon): Represents simple arrows.

ArcSrite (Arc, BaseSprite): Does not define any additional attributes.

ArrowSprite (Arrow, BaseSprite): Does not define any additional attributes.

CircleSprite (Circle, BaseSprite): Does not define any additional attributes.

EllipseSprite (Ellipse, BaseSprite): Does not define any additional attributes.

PolygonSprite (Polygon, BaseSprite): Does not define any additional attributes.

Constants

MIDDLE, TAIL, TIP: Values for specifying the anchor point for the Arrow class.
