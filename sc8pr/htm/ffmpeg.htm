sc8prx.ffmpeg Module

This module defines classes for encoding and decoding media files with FFmpeg using the imageio and imageio-ffmpeg packages. These packages, their requirements (numpy and Pillow), and FFmpeg must all be available to use these classes.

Classes

Reader: A class for decoding FFmpeg-compatible movies using imageio and imageio-ffmpeg.

Movie (CostumeImage): A class for playing FFmpeg-compatible movies in the sketch.

MovieSprite (Movie, BaseSprite): A class for playing FFmpeg-compatible movies in the sketch.

Writer: A class for encoding FFmpeg-compatible movies using imageio and imageio-ffmpeg.
