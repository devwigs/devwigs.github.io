Tutorial

This tutorial is intended for programmers with a basic knowledge of Python 3. You should be comfortable with:

  • importing modules,
  • defining your own functions,
  • using local variables,
  • operators (arithmetic, comparison, logical),
  • if, elif, else and while statements,
  • accessing object attributes using “dot notation”.

If you are new to Python programming, please visit Python For Beginners.

No advanced knowledge of data structures or object-oriented programming is required for this tutorial. Occassionally, a page will contain information for advanced programmers; these pages will contain a warning at the top of the page. Novice programmers can skip over these pages without harm.

Please ensure that you have installed all required software before starting the tutorial.

This tutorial assumes you are using the latest relase (version 2.2b) of sc8pr.
