sc8prx.pil.Grabber Class

The Grabber class performs screen captures using the Pillow package. The example below captures a portion of the screen starting at coordinates (100, 100), converts it to a sc8pr.Image instance, and adds the image to the sketch...

from sc8prx.pil import Grabber
from sc8pr import Sketch

def setup(sk):
    w, h = sk.size
    sk += Grabber([100, 100, w, h]).img.config(pos=sk.center)

Sketch().play()

Constructor

Grabber(rect=None)

rect: The region of the screen to capture. Can be None to capture the entire screen, a 2-tuple or 2-list specifying the size of the rectangle to capture from the upper left, or a 4-list or 4-tuple giving both the upper left coordinates and the size.

Read-only Properties

img: The current contents of the specified rectangle as an Image instance.

pil: The current contents of the specified rectangle as a PIL.Image.Image instance. This is the quickest format for screen captures.

rgba: The current contents of the specified rectangle as an Image instance that has been converted to support transparency.

srf: The current contents of the specified rectangle as an pygame.Surface instance.
