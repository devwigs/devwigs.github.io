sc8pr.shape.Arrow Class

The Arrow class is used for drawing simple arrow shapes. Arrow defines its own constructor, but otherwise inherits all of the attributes of the Polygon class...

Constructor

Arrow(length, width=0.1, head=0.1, flatness=2): Creates an arrow of the specified length. You can call the config method to set the pos and angle properties to control the arrow’s location and orientation. The tip of the arrow is used as the anchor property by default.

length: The length in pixels of the arrow from tip to tail.

width: The width of the arrow as a proportion of its length.

head: The length of the arrowhead as a proportion of the arrow’s total length.

flatness: Specifies the shape of the arrowhead.

Static Method

between(tail, tip, width=0.1, head=0.1, flatness=2) → Arrow: Create an arrow instance from tail to tip.

tail, tip: Coordinates of the tail and the tip of the arrow.

width, head, flatness: Same as for the constructor.
