sc8pr.shape.Arrow Class

The Arrow class is used for drawing simple arrow shapes. The shape may be distorted if the canvas has an attached coordinate system with different scales for the x and y directions. Arrow defines its own constructor, but otherwise inherits all of the attributes of the Polygon class...

Constructor

Arrow(**kwargs): Create an arrow shape. As of v2.2, all arguments are passed as keyword arguments. The previous version of this function is described below.

tail: The position of the arrow tail; default (0, 0).

tip: The position of the arrow tip; if omitted it will default to (length, 0).

length: The length of the arrow; theis argument is ignored if the tip is specified.

relative: specifies whether the width and head arguments are relative to the length of the arrow or absolute; default True.

width: The width of the arrow tail either in canvas coordinate system units or as a proportion of its length; default 1/14.

head: The length of the arrowhead either in canvas coordinate system units or as a proportion of its length; default 2/7.

angle: The angle of the arrowhead; default 35.

shape: Omit to use the default shape, or set to one of the constants SHARP or SIMPLE.

anchor: One of the constants TIP (default), TAIL, or MIDDLE, specifying which point to use as the polygon anchor point; can also be set to an ordered pair specifying another point.

Arrow(length, width=0.1, head=0.1, flatness=2): Creates an arrow of the specified length. You can call the config method to set the pos and angle properties to control the arrow’s location and orientation. The tip of the arrow is used as the anchor property by default. [Removed in v2.2]

length: The length in pixels of the arrow from tip to tail.

width: The width of the arrow as a proportion of its length.

head: The length of the arrowhead as a proportion of the arrow’s total length.

flatness: Specifies the shape of the arrowhead.

Properties

middle: The coordinates of the arrow middle.

tail: The coordinates of the arrow tail.

tip: The coordinates of the arrow tip.

Static Method

between(tail, tip, width=0.1, head=0.1, flatness=2) → Arrow: Create an arrow instance from tail to tip. [Removed in v2.2]

tail, tip: Coordinates of the tail and the tip of the arrow.

width, head, flatness: Same as for the constructor.
