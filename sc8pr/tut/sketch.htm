A sketch is an interactive animation that plays in the Pygame window. The name sketch is a shortening of “sketchbook” which is what such animations are called in the Processing programming language.

Here is a Python 3 program to create a minimal sketch that displays nothing but a black background...

from sc8pr.sketch import Sketch

Sketch().play()

The Sketch constructor creates a sketch object, and its play method plays the sketch in the Pygame window. To add content to the sketch, we need to provide a setup function...

from sc8pr.sketch import Sketch, Sprite, WRAP

def setup(sk):
    sk.setBackground("bg.png")
    Sprite(sk, "costume.png", spin=1, velocity=(2,1), edge=WRAP)

Sketch(setup).play()

The setup function takes a single argument, which is a reference to the sketch object. This gives the setup function access to the sketch properties such as its size.

In this example, the setup function loads an image from a file and sets it as the sketch background using the setBackground method. It then loads a second image file and uses it to create a sprite within the sketch. Sprites are characters or objects that can interact with each other and that the user can interact with using the mouse or keyboard. The Sprite constructor accepts a number of optional keyword arguments that describe the behaviour of the sprite. The Sprite class defined in sc8pr.sketch does not use Pygame’s sprite module.

Note that in the final line of the program, the setup function is passed as the lone argument to the Sketch constructor.
