In sc8pr, interactive animations are called sketches. This term is shorted from 'sketchbook', which is what animations are called in the Processing programming language. To run an animation, you will create a sketch and then play it...

from sc8pr import Sketch

Sketch().play()

You can specify the initial size of your sketch by passing an ordered pair to Sketch. If you do not specify the size, it will default to 512 pixels wide by 288 pixels tall.

size = 640, 480
Sketch(size).play()

You can also provide up to three optional arguments to the play method, to specify the animation window’s caption and icon. The third argument is a boolean value indicating whether to allow the user to resize the window; this defaults to True.

caption = "An Awesome Game"
icon = "icon.png"
mode = False
Sketch().play(caption, icon, mode)
