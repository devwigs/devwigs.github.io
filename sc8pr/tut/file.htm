The Sketch class includes a fileDialog method that allows the user to browse for files or folders. File dialogs work much the same as user prompts in that they run asynchronously and they generate a USERINPUT event when the dialog is closed...

from sc8pr.sketch import Sketch, USERINPUT, OPEN

def setup(sk):
    sk.setBackground("bg.png")
    sk.fileDialog(OPEN, initFilter="<Image>")
    sk.animate(eventMap={USERINPUT:fileSelected})

def fileSelected(sk, ev):
    print(ev.value)
    sk.quit = True

Sketch(setup).play((800,600))

The first argument to the fileDialog method is one of the constants OPEN, SAVE, or FOLDER which specify the purpose of the dialog. The other arguments are allowCancel (which indicates whether the Cancel button is enabled or disabled), and initFilter (which indicates the types of files to display). The value of initFilter can be a pattern (e.g. "*.png" or "*.png;*.jpg") or a group of file types (e.g. "<Image>;<Video>")

File types and their groups and icons are defined in the "icons" subfolder of the sc8pr package and in the file "fileBrowser.json" located in that folder. To customize the icons or file types, you can set the iconPath and iconJson attributes of the FileBrowser class...

from sc8pr.grid import FileBrowser

FileBrowser.iconPath = "myIcons/{}.png"
FileBrowser.iconJson = "myIcons/myIcons.json"
