One of the most important events is the “draw” event. This event is triggered each time a graphics object is drawn, i.e. once per frame for each object. In the program below, this event will be triggered each frame for the score text object first, then for the alien sprite, and finally for the sketch itself. (The draw event does not occur for the background image.)

from sc8pr import Sketch, Image, TOPLEFT
from sc8pr.sprite import Sprite
from sc8pr.text import Text

def setup(sk):
    sk.bg = Image("sky.png")
    score = Text(0)
    score.bind(ondraw = countFrame)
    score.config(
        pos = (8, 8),
        anchor = TOPLEFT,
        color = "red",
        fontSize = 32)
    alien = Sprite("aliens.png", 2, 2)
    alien.config(
        pos = sk.center,
        width = sk.width/10,
        costumeTime = 10)
    sk += score, alien

def countFrame(score, sk):
    score.config(data = sk.frameCount)

Sketch((512,362)).play()

This example provides an event handler for score only. During the setup, we call the object’s bind method to specify which function (i.e. countFrame) will serve as the event handler. The event handler takes two arguments: references to the graphics object itself and to the sketch. In this example, the event handler reconfigures the text so that it displays the current frame number rather than its original value which was the number 0.

If we choose to, we can bind different functions to handle the draw event for the alien and for the sketch itself.
