The example program on the previous page created a sketch that displayed a white background. If you want to change the background or add more content to the sketch, you need to create a setup function. This function takes a single argument, which refers to the sketch object. In the example below, the setup function changes the sketch’s background color to blue...

from sc8pr import Sketch

def setup(sk):
    sk.bg = "blue"

Sketch().play()

Colors can be described by any color name that pygame recognizes, or by an RGB tuple or hexadecimal string. The following statements are alternative ways of describing the color blue...

sk.bg = 0, 0, 255
sk.bg = "#0000ff"

Your sketch background does not have to be a single color. You can load an image to use as your background instead...

from sc8pr import Sketch, Image

def setup(sk):
    sk.bg = Image("bg.png")

Sketch().play()

If the image aspect ratio does not match the aspect ratio of the sketch, the sketch will be resized so that the background can be displayed without being stretched.
