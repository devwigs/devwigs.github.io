The example program on the previous page created a sketch that displayed a white background. If you want to change the background or add more content to the sketch, you need to create a setup function. This function takes a single argument, which refers to the sketch object. In the example below, the setup function changes the sketch’s background color to blue...

from sc8pr import Sketch

def setup(sk):
    sk.bg = "blue"

Sketch().play()
