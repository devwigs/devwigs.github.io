You can use your setup function to add graphics objects to your sketch. This can be done using the sketch’s += operator...

from sc8pr import Sketch, Image
from sc8pr.sprite import Sprite
from sc8pr.text import Text

def setup(sk):
    sk.bg = Image("bg.png")
    sk += Sprite("ship.png")
    sk += Text("Hello, world!")

Sketch().play()
