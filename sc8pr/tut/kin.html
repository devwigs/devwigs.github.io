Sprites also have the ability to move. In the example below, we have added a spin argument when configuring the sprite to make it rotate 1 degree clockwise per frame. We have also added a vel (velocity) argument to move the sprite 2 pixels right and 1 pixel down each frame...

from sc8pr import Sketch, Image, TOPLEFT, VERTICAL, HORIZONTAL
from sc8pr.sprite import Sprite
from sc8pr.text import Text

def setup(sk):
    sk.bg = Image("sky.png")
    score = Text(0)
    score.config(
        pos = (8, 8),
        anchor = TOPLEFT,
        color = "red",
        fontSize = 32)
    alien = Sprite("aliens.png", 2, 2)
    alien.config(
        pos = sk.center,
        width = sk.width/10,
        costumeTime = 10,
        spin = 1,
        vel = (2, 1),
        bounce = HORIZONTAL,
        wrap = VERTICAL)
    sk += score, alien

Sketch((512,362)).play()

The bounce and wrap arguments determine what happens when the sprite reaches the edge of the sketch. By specifying bounce = HORIZONTAL, when the sprite hits the left or right sides it will bounce back the way it came. By setting wrap = VERTICAL, the sprite will disappear from the top or bottom and reappear from the opposite edge. If we want the same behaviour from all four edges, we could use BOTH rather than HORIZONTAL or VERTICAL. The wrap argument can also take values REMOVE, REMOVE_X, or REMOVE_Y, to indicate that the sprite should be removed from the animation if it completely leaves the sketch area.
