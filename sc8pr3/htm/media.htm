sc8pr.misc.media Module

This module contains additional classes for encoding and decoding media files with FFmpeg using the imageio and imageio-ffmpeg packages. These packages, their requirements (numpy and Pillow), and FFmpeg must all be available to use these classes.

Classes

Grabber: A class for performing screen captures using the Pillow package.

FFReader: A class for decoding movies using FFmpeg.

FFWriter: A class for encoding movies using FFmpeg.

Movie (Image): An class for playing movies as they are decoded from an S8V or FFmpeg-compatible file.

MovieSprite (Movie, BaseSprite): A sprite subclass of Movie.
