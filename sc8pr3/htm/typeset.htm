sc8pr.misc.typeset Module

Experimental!

This module can be used for arranging text into paragraphs and lines.

from sc8pr import Sketch
from sc8pr.text import Font, ITALIC
from sc8pr.misc.typeset import typeset

TEXT = """You will rejoice to hear that no disaster has accompanied the commencement of an enterprise which you have regarded with such evil forebodings. I arrived here yesterday; and my first task is to assure my dear sister of my welfare, and increasing confidence in the success of my undertaking.
I am already far north of London; and as I walk in the streets of Petersburgh, I feel a cold northern breeze play upon my cheeks, which braces my nerves, and fills me with delight. Do you understand this feeling? This breeze, which has travelled from the regions towards which I am advancing, gives me a foretaste of those icy climes...
Frankenstein, Mary Shelley (1831)"""

def setup(sk):
    cv = typeset(TEXT, 640, 6, 14, font=Font.serif(), fontSize=16)
    cv[-1].config(fontStyle=ITALIC)
    sk.size = cv.size
    sk += list(cv)

Sketch().play(caption="Frankenstein")

Functions

breakLine(text, font, width, chars=" -") → str: Insert new-line (\n) characters into a string to fit the displayed text to the desired width.

text: A string to which line breaks will be added; should not contain any new-line characters.

font: A pygame.font.Font instance representing the font renderer; this can be the renderer property of any Text instance or the result of a call to the Font.get method.

width: The desired width of the rendered paragraph in pixels; if too small, lines will be broken at the smallest possible width based on the text and font.

chars: A sequence of characters after which line breaks can be placed.

breakLines(text, font, width, chars=" -") → list: Add line breaks to one or more paragraphs and return a list of strings.

text: One or more paragraphs as a string delimitted by \n characters, or as a list or tuple of strings.

font, width, chars, : See breakLine function.

typeset(text, width, padding=0, spaceBetween=None, strictWidth=False, chars=" -", **kwargs) → Canvas: Arrange paragraphs into a canvas.

text, width, chars, : Arguments to be passed to the breakLines function.

padding: Pixels to be added around the outside of the canvas.

spaceBetween: Pixels to be added between paragraphs; if omitted this will default to three times the line spacing as set by the spacing keyword argument.

strictWidth: A bool specifying whether the canvas width should match the requested width plus padding exactly (True) or whether it can be enlarged or reduced to fit the rendered text (False).

kwargs: Additional arguments to be passed to the Text constructor, which may include font, fontSize, fontStyle, spacing, and align; the anchor property should not be specified as it will be set based on the align property to produce the appropriate layout of Text instances within the canvas .
