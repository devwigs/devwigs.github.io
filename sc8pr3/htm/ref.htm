sc8pr 3 Reference

Essentials

  • The sc8pr module defines essential classes such as Sketch, Image, and Canvas.
  • The sc8pr.geom module defines various 2D geometry-related functions.
  • The sc8pr.util module defines several helper and general purpose functions.

Common

  • The sc8pr.sprite module defines the Sprite class for creating interactive, animated graphics.
  • The sc8pr.shape module defines the Line, Circle, Polygon, and Arrow classes and their corresponding sprites.
  • The sc8pr.text module defines the Text and Font classes for adding text to your sketch.

Extras

  • The sc8pr.gui package defines additional graphics classes for creating Graphical User Interface controls.
  • The sc8pr.misc package defines additional features such as video effects and classes for importing and exporting video.
  • The sc8pr.plot package defines additional features for plotting data.
  • The sc8pr.robot module defines features for creating robotics simulations.
