sc8pr.gui.dialog.Dialog Class

The Dialog class extends the Canvas class by adding three instance methods that may be useful in composing dialog boxes for interacting with the user.

In addition to the attributes it inherits from Canvas, the Dialog class defines the following...

Instance Methods

resize() → None: Overrides Canvas.resize so that the dialog is not resized along with the sketch. If you want the dialog to resize with the sketch, call dlg.bind(resize=Canvas.resize) where dlg refers to the Dialog instance, or create a subclass defining the resize attribute as Canvas.resize.

title(title, padding=4, **kwargs) → self: You can add a title bar manually be calling this method rather than passing a title to the constructor. This is useful if you want to add custom content to the dialog.

title: The title as a string.

padding: The number of pixels of space to be added around the title bar.

kwargs: Additional arguments such as font or fontSize that will be used to configure the Text and TextInput items.

insertTop(gr, padding=12, name=None) → self: Inserts a graphic into the top of the dialog and shifts all existing content down accordingly.

gr: The graphic to insert.

padding: Additional vertical space to insert, in pixels.

name: The name (key) to use when adding the graphic to the dialog.
