sc8prx Package

sc8prx defines extra features for sc8pr that have additional dependencies:

  • numpy
  • Pillow
  • imageio
  • imageio-ffmpeg
  • matplotlib

sc8prx must be installed separately from sc8pr. As sc8pr 3 is still in pre-release, it needs to be installed from GitHub, but will be available on PyPI when it is officially released.

pip3 install -U https://github.com/dmaccarthy/sc8prx/archive/refs/heads/main.zip
