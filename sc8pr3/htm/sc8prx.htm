sc8prx Package

sc8prx defines extra features for sc8pr that have additional dependencies:

  • numpy
  • Pillow
  • imageio
  • imageio-ffmpeg
  • matplotlib

sc8prx must be installed separately from sc8pr. As sc8pr 3 is still in pre-release, you will need to install sc8prx from GitHub after installing the core sc8pr package first. When version 3 is officially released, both packages will be available on PyPI and installing sc8prx will install all dependencies including the core sc8pr.

pip3 install -U https://github.com/dmaccarthy/sc8prx/archive/refs/heads/main.zip

Function

pil(pix) → PIL.Image.Image: Convert the image format from sc8pr.PixelData to a Pillow format image.

pix: A PixelData instance.
