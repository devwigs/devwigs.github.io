sc8pr.gui.textinput.TextInputCanvas Class

The TextInputCanvas class is Canvas subclass that contains a TextInput control. In addition to the attributes it inherits from Canvas, the TextInputCanvas class defines the following...

Constructor

TextInputCanvas(width=None, data="", prompt=None, center=False, vertical=False, **text)

width: The width of the canvas in pixels.

data: A str describing the text input content.

prompt: A str to be displayed when the text input data is empty.

center: A bool specifying whether the text should be centre-aligned.

vertical: A bool specifying whether the text should be displayed vertically or horizontally.

**text: Arguments specifying the text appearance such as font or color.

Assignable Properties
