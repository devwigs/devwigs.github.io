sc8pr.misc.movie.Movie Class

The Movie class is used to play media files as they are decoded. This class uses the imageio and imageio-ffmpeg packages as well as their dependencies (numpy, Pillow) to decode FFmpeg-compatible files.

from sc8pr import Sketch
from sc8pr.misc.movie import Movie

def setup(sk):
    mov = Movie("movie.mkv")
    sk.size = mov.size
    sk += mov.config(pos=sk.center, costumeTime=1)

Sketch().play()

Constructor

Movie(src, skip=0, frames=None, alpha=False, **kwargs): .

src: The path to an FFmpeg-compatible media file to be decoded.

skip: The number of frames to skip at the beginning of the movie.

frames: The maximum number of frames to read. If the requested number of frames is more than available, the clip property and len function may return incorrect values until the movie has been completely read.

alpha: A bool specifying whether the loaded frames should be converted to support transparency.

kwargs: Additional options to be passed to imageio.get_reader, incldung size.

Read-Only Properties

clip: A 2-tuple specifying the starting and ending frame numbers. The ending frame number may be None until the movie has been completely read.

costumeList: Returns None as this feature is not supported by Movie.

costumeSequence: Returns None as this feature is not supported by Movie.

cycle: Returns False as this feature is not supported by Movie.

Instance Method

close() → self: Close the FFReader instance that is decoding the movie.

restart() → self: Restarts the movie by closing the reader and creating a new FFReader instance using the same arguments as the original one.
