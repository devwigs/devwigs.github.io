GUI Controls

The sc8pr.gui subpackage defines several types of graphics objects that can be used to interact with the user. The example below shows how to use a TextInputCanvas (a Canvas that contains a TextInput control) to obtain text data from the user asynchronously.

TextInputCanvas instances trigger onchange and onaction events. Your program can bind handlers to these events to access the data...

from sc8pr import Sketch
from sc8pr.text import Font
from sc8pr.gui.textinput import TextInputCanvas

# Event handlers
def onchange(gr, ev): print("Change:", gr.data)
def onaction(gr, ev): print("Action:", gr.data)

def setup(sk):
    # Create the TextInput within a Canvas
    style = dict(color="blue", font=Font.sans(), padding=4, promptColor="red")
    name = TextInputCanvas(360, "", "Enter Your Name", **style)

    # Configure the TextInputCanvas
    name.config(pos=sk.center, bg="#f0f0f0", weight=1)

    # Bind event handlers and add to the sketch
    sk += name.bind(onchange, onaction)

# Create and run the sketch
Sketch((384, 144)).play()

As with other types of graphics, you do not need to use the event names (onchange, onaction) as the names of the event handler functions. If you choose to use a different name, pass the handler to the bind method as a keyword argument rather than a positional one. For example, if you want to name the onaction handler submitName...

name.bind(onchange, onaction=submitName)
