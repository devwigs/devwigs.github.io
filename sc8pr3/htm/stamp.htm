sc8pr.effect.stamp Module

This module defines additional effects that modify an image by blitting another image on top of it (like an ink stamp).

Base Class

Stamp: This is the abstract base class for stamp effects. This class overrides Effect.apply with a method that applies the stamp. Subclasses must provide a get_stamp(t, size) method that returns a surface to use as the stamp. The argument t is a value between 0 and 1 that specifies the transition timing, and size is the size of the surfaces.

Assignable Property

special_flags: The value of the special_flags argument to be passed to pygame.Surface.blit when the images are combined; default pygame.BLEND_RGBA_MIN.

Static Method

new_surface(size) → pygame.Surface: Creates a new surface of the specified size that has been filled with the RGBA value (0, 0, 0, 0).

Subclass Constructors

Pupil(): Defines a get_stamp method to stamp a circle or regular polygon over the image where the exterior of the shape is transparent and the interior is the specified fill color. Instances have the following additional properties that can be set by calling the config method:

center: The coordinates of the circle or polygon’s geometric centre relative to the size of the image. By default, this will be (0.5, 0.5) which centres the figure at the centre of the image.

sides: The number of sides for a regular polygon or None (default) for a circle.
