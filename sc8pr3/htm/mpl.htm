sc8prx.mpl Module

Experimental!

This module defines functions for converting matplotlib figures to sc8pr images. For example, you can use matplotlib to render equations as sc8pr images.

from sc8pr import Sketch
from sc8prx.mpl import text, fonts

def setup(sk):
    fonts("cm")
    formula = r"$V = \frac{4\pi}{3} r^3$"
    sk += text(formula, "red").config(pos=sk.center)

Sketch().play()

Functions

figure(fig, image=True, **kwargs) → Image or bytes: Renders a matplotlib figure as an Image instance or as a bytes object containing the image data in PNG format.

fig: A matplotlib figure.

image: Specifies whether to return the result as a sc8pr.Image.

**kwargs: Additional arguments to be passed to the figure’s savefig method, which may include dpi (default 300), transparent (default True), bbox_inches (default "tight"), and pad_inches (default 0.025).

fonts(math="stix", sans=None, serif=None, mono=None) → None: Specify font names for matplotlib to use for text/math rendering.

text(text, color="black", fontsize=12, image=True, **kwargs) → Image or bytes: Create a text/math figure and then pass it to the figure function described above.

text: The text to be rendered; math can be included enclosed between a pair of $ symbols.

color: The color in which the text is to be rendered.

fontsize: The font size in which the text is to be rendered.

image, **kwargs: Arguments to be passed to the figure function.
