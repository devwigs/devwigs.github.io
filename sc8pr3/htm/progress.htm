sc8pr.misc.progress.ProgressBar Class

The ProgressBar class is a Canvas subclass that can be used to display a simple progress bar graphic.

Usage example...

from sc8pr import Sketch
from sc8pr.misc.progress import ProgressBar

def setup(sk):
    prg = ProgressBar(color="blue").bind(ondraw)
    sk += prg.config(pos=sk.center, weight=1)

def ondraw(prg, ev=None):
    frame = prg.sketch.frameCount
    prg.value = min(1, frame / 600)

Sketch().play()

Constructor

ProgressBar(size=(128, 16), color="gray", lower=0, upper=1)

size: The size of the canvas.

color: The color of the filled region.

lower: The value corresponding to minimum progress.

upper: The value corresponding to maximum progress.

Assignable Property

value: A value specifying how much of the progress bar to fill. Default is the lower value passed to the constructor.
