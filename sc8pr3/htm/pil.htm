sc8prx.pil Module

This module enables the use of Pillow-formatted images in sc8pr.

Classes

Grabber: A class for performing screen captures using Pillow.

Functions

pil_to_surface(pil_img) → pygame.Surface: Converts a PIL image to a surface.

pil_img: A PIL.Image.Image to convert.

surface_to_pil(srf, mode=None) → PIL.Image.Image: Converts an Image, pygame.Surface (or any object that can be converted to a surface) to a PIL image.

srf: The image to convert.

mode: Either "RGBA" or "RGB"; if omitted the format of the original image will be used.
