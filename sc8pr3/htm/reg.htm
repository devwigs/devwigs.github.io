sc8pr.plot.regression Module

Experimental!

This module defines functions for performing simple regressions.

Functions

expon(x, y) → (lambda, (float, float)): Perform an exponential regression and return the model (y = abx) as a lambda, and a tuple containing the values of a and b.

x: A list or tuple containing the x data.

y: A list or tuple containing the y data.

leastSq(x, y) → (lambda, (float, float)): Perform a least-squares linear regression and return the model (y = mx + b) as a lambda, and a tuple containing the values of m and b. Arguments are the same as for expon.

power(x, y) → (lambda, (float, float)): Perform a power regression and return the model (y = axn) as a lambda, and a tuple containing the values of a and n. Arguments are the same as for expon.
