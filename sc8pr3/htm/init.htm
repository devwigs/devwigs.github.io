sc8pr Module

Classes

Graphic: Abstract base class for graphics instances that can be added to the sketch or any canvas.

Image (Graphic): Class representing static bitmap images.

Canvas (Graphic): A rectangular drawing region that can contain other graphics instances (including other canvases.)

Sketch (Canvas): A top-level canvas that knows how to animate itself using pygame.

BaseSprite (Graphic): Abstract base class for various types of sprites.

Renderable (Graphic): Abstract base class for graphics that are drawn using a render method.

PixelData: A class for representing images as raw binary data. Used for data compression and format conversion.

Constants

version: A 3-tuple describing the sc8pr version, such as (3, 0, 0). Non-production versions will have a string such as "a2" (alpha), or "b0" (beta) as the final item of the tuple.

CENTER, TOP, BOTTOM, LEFT, RIGHT, TOPLEFT, TOPRIGHT, BOTTOMLEFT, BOTTOMRIGHT: Constants representing possible values for the anchor attribute of Graphic instances.

HORIZONTAL, VERTICAL, BOTH: Constants representing possible values for the bounce and wrap attributes of BaseSprite instances.

REMOVE, REMOVE_X, REMOVE_Y: Constants representing additional values for the wrap attribute of BaseSprite instances.

CIRCLE, RECT: Constants representing the available models for bouncing sprites from the canvas edge.
