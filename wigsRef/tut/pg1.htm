A sketch is an interactive animation that plays in the Pygame window. The name sketch is a shortening of “sketchbook”, which is what such animations are called in the Processing programming language.

Here is a Python 3 program to create a minimal sketch that displays nothing but a black background...

from wigs.sketch import Sketch

Sketch().run()

The Sketch constructor (from the wigs.sketch module) creates a sketch object, and its run method runs the sketch in the Pygame window.

To add content to the sketch, we need to provide a setup function. The setup function takes a single argument, which is a reference to the sketch object...

from wigs.sketch import Sketch, Sprite, BOUNCE

def setup(sk):
    sk.setBackground("bg.png")
    Sprite(sk, "costume.png", spin=1, posn=sk.center, velocity=(2,1), edge=BOUNCE)

Sketch(setup).run()
