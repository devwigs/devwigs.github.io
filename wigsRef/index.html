Overview

wigs is a Python 3 package for educational programming. Its aim is to help beginning (and experienced!) programmers make interactive programs with graphics and animations easily.

Features include:

  • a simple framework for creating interactive animations
  • image manipulation
  • sprite management
  • graphical user interface
  • robotics simulations

It requires only a few lines of code to create a simple animation...

from wigs.sketch import Sketch, Sprite

def setup(sk):
    sk.setBackground("bg.png")
    Sprite(sk, "costume.png", posn=sk.center, spin=1)

Sketch(setup).run()

Installation Guide

The wigs package requires:

  • Python 3.2+
  • Pygame 1.9.1+

Please ensure that Python and Pygame are both installed before setting up wigs.

Download the wigs source code from GitHub as a zip or tar.gz archive. Open the archive with your favourite archive program* and extract the wigs package. (*Windows Explorer can be used to open the zip format archive.)

As wigs is entirely Python 3 source code, no special installation is required. Simply extract the wigs folder to your source folder, or to another location in your PYTHONPATH (e.g. the Lib/site-packages folder of your Python installation.)
