Overview

j5tools is a collection of Javascript tools, primarily for HTML5 documents. Many of the features depend on jQuery. The repository contains individual Javascript files (listed below) as well as a single minified file, j5tools-min.js, containing all of the individual files.

JavascriptDescription
plot.jsCanvas plotting tools.
util.jsVarious Javascript utilities.
