Practice

    1. A vertical spring with \(k\) = 850 N/m compresses 2.80 cm (at equilibrium) when an object is placed on top of it. Determine the weight and mass of the object. [hint: At equilibrium, the elastic and gravity forces are balanced!]
      2. $$\vec{\mathbf F}_{elas} = -k\Delta L = \mathrm{-850\ N/m\cdot (-0.0280\ m) = +23.8\ N}$$ Since the elastic force is 23.8 N up, the force of gravity (weight) is 23.8 N down because the forces are balanced at equilibrium. $$m = \frac{F_g}{g} = \mathrm{\frac{23.8\ N}{9.81\ m/s^2} = 2.43\ kg}$$
    2. If the object is set in motion vertically, what will its period be?
      3. $$T = 2\pi\sqrt{\frac{m}{k}} = \mathrm{2\pi\sqrt{\frac{2.43\ kg}{850\ N/m }} = 0.336\ s}$$
  2. A vertical spring stretches 5.20 cm when a 63.0 g mass is attached to its end.
    1. What is the elastic constant of the spring?
      2. $$F_{elas} = F_g = mg = \mathrm{0.0630\ kg\cdot 9.81\ m/s^2 = 0.618\ N}$$ $$k = \frac{F_{elas}}{\Delta L} = \mathrm{\frac{0.618\ N}{0.0520\ m} = 11.9\ N/m}$$
    2. If the 63.0 g mass is replaced by a 105 g mass, what will the total stretch of the spring be?
      3. $$F_{elas} = F_g = mg = \mathrm{0.105\ kg \cdot 9.81\ m/s^2 = 1.03\ N}$$ $$\Delta L = \frac{F_{elas}}{k} = \mathrm{\frac{1.03\ N}{11.9\ N/m} = 0.0867\ m}$$
    3. Calculate the period of the oscillations for both masses.
      4. $$T_1 = 2\pi\sqrt{\frac{m_1}{k}} = \mathrm{2\pi\sqrt{\frac{0.0630\ kg}{11.9\ N/m }} = 0.457\ s}$$ $$T_2 = 2\pi\sqrt{\frac{m_2}{k}} = \mathrm{2\pi\sqrt{\frac{0.105\ kg}{11.9\ N/m }} = 0.590\ s}$$
    1. A spring with \(k\) = 65.0 N/m is attached to a 95.0 g mass. Predict the period and frequency of the SHM when the mass is set in motion.
      2. $$T = 2\pi\sqrt{\frac{m}{k}} = \mathrm{2\pi\sqrt{\frac{0.0950\ kg}{65.0\ N/m }} = 0.240\ s}$$ $$f = \frac{1}{T} = \mathrm{\frac{1}{0.240\ s} = 4.16\ Hz}$$
    2. Calculate the maximum velocity and acceleration of the mass if its amplitude is 1.50 cm.
      3. $$v_{max} = \frac{2\pi A}{T} = \mathrm{\frac{2\pi\cdot 0.0150\ m}{0.240\ s} = 0.392\ m/s}$$ $$a_{max} = \frac{4\pi^2 A}{T^2} = \mathrm{\frac{4\pi^2\cdot 0.0150\ m}{(0.240\ s)^2} = 10.3\ m/s^2}$$
    1. A 275 g mass attached to the end of spring executes SHM with a period of 0.950 s. Calculate the \(k\) value of the spring.
      2. $$T = 2\pi\sqrt{\frac{m}{k}}$$ $$T^2 = \frac{4\pi^2 m}{k}$$ $$kT^2 = 4\pi^2 m$$ $$k = \frac{4\pi^2 m}{T^2} = \mathrm{\frac{4\pi^2\cdot 0.275\ kg}{(0.950\ s)^2} = 12.0\ N/m}$$
    2. If the mass has a maximum velocity of 0.580 m/s, determine its amplitude.
      3. $$v_{max} = \frac{2\pi A}{T}$$ $$v_{max}T = 2\pi A$$ $$A = \frac{v_{max}T}{2\pi} = \mathrm{\frac{0.580\ m/s\cdot0.950\ s}{2\pi} = 0.0877\ m}$$
    3. Draw accurate graphs of the object’s position, velocity, and acceleration as functions of time.
      4. $$a_{max} = \frac{4\pi^2 A}{T^2} = \mathrm{\frac{4\pi^2\cdot 0.0877\ m}{(0.950\ s)^2} = 3.84\ m/s^2}$$

  5. A spring has \(k\) = 32.0 N/m. What mass would need to be attached to the spring to produce SHM with a period of 1.00 s?
    6. $$T = 2\pi\sqrt{\frac{m}{k}}$$ $$T^2 = \frac{4\pi^2 m}{k}$$ $$kT^2 = 4\pi^2 m$$ $$m = \frac{kT^2}{4\pi^2} = \mathrm{\frac{32.0\ N/m\cdot (1.00\ s)^2}{4\pi^2} = 0.811\ kg}$$
