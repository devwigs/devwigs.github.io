Review

  1. Describe in detail how to isolate an unknown variable in an equation.

Practice

  1. Given \(v = \mathrm{343\ m/s}\) and \(\lambda = \mathrm{0.825\ m}\), solve for \(f\) in the equation: $$v = f\lambda$$
    $$\frac{v}{\lambda} = f$$ $$f = \mathrm{\frac{343\ m/s}{0.825\ m} = 416\ s^{-1}}$$
  2. Given \(a = \mathrm{9.81\ m/s^2}\) and \(r = \mathrm{15.0\ m}\), solve for \(v\) in the equation: $$a = \frac{v^2}{r}$$
    $$ar = v^2$$ $$\sqrt{ar} = v$$ $$v = \mathrm{\sqrt{9.81\ m/s^2\cdot 15.0\ m} = 12.1\ m/s}$$
  3. Given \(g = \mathrm{9.81\ m/s^2}\) and \(T = \mathrm{0.750\ s}\), solve for \(L\) in the equation: $$T = 2\pi\sqrt{\frac{L}{g}}$$
    $$\frac{T}{2\pi} = \sqrt{\frac{L}{g}}$$ $$\left(\frac{T}{2\pi}\right)^2 = \frac{L}{g}$$ $$g\left(\frac{T}{2\pi}\right)^2 = L$$ $$L = \mathrm{9.81\ m/s^2\left(\frac{0.750\ s}{2\pi}\right)^2 = 0.140\ s}$$
  4. Given \(\vec{\mathbf a} = \mathrm{2.00\ m/s^2}\), \(\Delta t=\mathrm{3.50\ s}\), and \(\Delta\vec{\mathbf d} = \mathrm{25.0\ m}\), solve for \(\vec{\mathbf v}_i\) in the equation: $$\Delta\vec{\mathbf d} = \vec{\mathbf v}_i\Delta t + \frac{1}{2} \vec{\mathbf a} \left(\Delta t\right)^2$$
    $$\Delta\vec{\mathbf d} - \frac{1}{2} \vec{\mathbf a} \left(\Delta t\right)^2 = \vec{\mathbf v}_i\Delta t$$ $$\frac{\Delta\vec{\mathbf d} - \frac{1}{2} \vec{\mathbf a} \left(\Delta t\right)^2}{\Delta t } = \vec{\mathbf v}_i$$ $$\vec{\mathbf v}_i = \mathrm{\frac{25.0\ m - \frac{1}{2} \left(2.00\ m/s^2\right) \left(3.50\ s\right)^2}{3.50\ s} = 3.64\ m/s}$$
