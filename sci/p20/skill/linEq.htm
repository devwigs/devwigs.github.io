  • Calculate the rise by subtracting the y-coordinates (the responding variable): $$\Delta V = V_2 - V_1 =\mathrm{2.0\ V - 11.4\ V = -9.4\ V}$$
  • Calculate the run by subtracting the x-coordinates (the manipulated variable): $$\Delta I = I_2 - I_1 = \mathrm{5.0\ A - 0.0\ A = 5.0\ A}$$
  • Calculate the slope by dividing:$$k = {\Delta V\over \Delta I} = \mathrm{{-9.4\ V\over 5.0\ A} = -1.9\ V/A}$$

\(y\)-Intercept

The \(y\)-intercept is the value where the line crosses the \(y\)-axis.

  • In this example, we can see from the graph that the intercept is 11.4 volts.
  • Sometimes, the intercept may not be visible on the graph.
  • You can calculate the intercept (\(b\)) using the equation:

$$y = kx + b$$

$$y - kx = b$$

$$b = \mathrm{2.0\ V - (-1.88\ V/A) 5.0\ A = 11.4\ V}$$

Model Equation

Once you know the slope and the \(y\)-intercept, you can write the equation in the form: \(y = kx+b\).

  • Replace \(x\) and \(y\) by the actual variable names or symbols.
  • Replace \(k\) and \(b\) by the calculated values, including units.

$$V = \mathrm{(-1.9\ V/A)} I + \mathrm{11.4\ V}$$

We can use the equation to make predictions about the data...

Example 1: Predict the voltage when the current is 3.5 A.

$$V = \mathrm{(-1.9\ V/A)\ \color{red}{3.5\ A} + 11.4\ V = 4.8\ V}$$

  • A prediction within the range of data is called interpolation.

Example 2: What current would be needed for the voltage to reach zero?

$$V - \mathrm{11.4\ V} = \mathrm{(-1.9\ V/A)} I$$

$${V - \mathrm{11.4\ V} \over \mathrm{-1.9\ V/A}} = I$$

$$I = {\color{red}{\mathrm{0.0\ V}} - \mathrm{11.4\ V} \over \mathrm{-1.9\ V/A}} = \mathrm{6.0\ A}$$

  • A prediction outside the range of data is called extrapolation.

Regression Analysis

Mathematicians have developed a technique called regression analysis that can calculate a model equation without relying on a graph.

  • Linear regression calculates the slope and \(y\)-intercept values that minimize how close the data points are to the best-fit line.
  • Many graphing calculators and data analysis software programs have regression abilities.
