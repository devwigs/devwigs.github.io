Assignment

  1. Implement a bubble sort algorithm by completing the function bubbleSort in the program iter3_sort.py. Do not use the operators < and > when comparing values as this will require different versions of the function to sort in ascending and descending order. Instead, the program stub provides two functions, ascend and descend, to perform comparisons. You can specify which one you want to use by assigning to the global variable compare. These functions return True when the arguments are passed in the incorrect order (and need to be swapped) or False when no swap is necessary.
