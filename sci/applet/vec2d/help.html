Help: 2D Vectors

To draw a vector, enter the components separated by a comma in the input area, and then type the Enter key...

To specify a vector in polar form, enter the magnitude followed by @, and then the direction in degrees...

To add two or more vectors, type the vectors (in either component or polar form) separated by semi-colons...

To adjust the view of the diagram, begin the input with ! and then enter the coordinates for the left, right, and bottom edges of the graph separated by commas. Do not enter the top coordinate as it will be calculated automatically to ensure that the axes have the same scale. If you omit the third value, the x-axis will drawn through the centre of the diagram. You can specify the grid interval as an optional fourth value.

When the input area is focussed, you can use the up and down arrows to cycle through the input history.

You can use the mouse to re-center the graph by dragging the vector diagram, and to zoom with the mouse wheel. Click the vector diagram to add a text label at the mouse location.

If you should “break” the applet with bad input, click the Reset button to recover.
