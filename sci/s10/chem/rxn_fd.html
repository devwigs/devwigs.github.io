Formation Reactions

A formation reaction is a reaction whereby two or more elements react to form a compound.

  • Formation reactions are also called synthesis or composition reactions.

Example: Write and balance the formation reaction for ammonia.

  • Start with the unbalanced reaction:$$\mathrm{N_2 + H_2 \rightarrow NH_3}$$
  • Balance the nitrogens:$$\mathrm{N_2 + H_2 \rightarrow 2\ NH_3}$$
  • Balance the hydrogens:$$\mathrm{N_2 + 3\ H_2 \rightarrow 2\ NH_3}$$
  • Include the phases if possible:$$\mathrm{N_{2(g)} + 3\ H_{2(g)} \rightarrow 2\ NH_{3(g)}}$$

Decomposition Reactions

A decomposition reaction is the reverse of a formation reaction...

...a compound breaks down into its separate elements.

Example: Write and balance the decomposition reaction for CH3Cl.

  • Start with the unbalanced reaction:$$\mathrm{CH_3Cl \rightarrow C + H_2 + Cl_2}$$
  • Balance the hydrogen first because it will be the hardest:$$\mathrm{2\ CH_3Cl \rightarrow C + 3\ H_2 + Cl_2}$$
  • The chlorine is already balanced!
  • Do carbon last because it will be the easiest:$$\mathrm{2\ CH_3Cl \rightarrow 2\ C + 3\ H_2 + Cl_2}$$
