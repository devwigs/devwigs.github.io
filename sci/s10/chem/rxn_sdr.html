Single Replacement Reactions

A single replacement reaction is when an element reacts with a compound to replace an element that is part of the compound: $$\mathrm{FeCl_3 + Ag \rightarrow Fe + AgCl}$$

  • In this unbalanced reaction, silver replaces iron in the compounds.
  • In ionic compounds, cations can only be replaced by cations and anions by anions.
  • Note that the subscript on chloride changes because the silver in has a different charge than iron(III).
  • The reaction needs to be balanced: $$\mathrm{FeCl_3 + 3\ Ag \rightarrow Fe + 3\ AgCl}$$

Example: Write and balance the SR reaction for oxygen reacting with calcium nitride.

  • Start with the unbalanced reaction:$$\mathrm{Ca_3N_2 + O_2 \rightarrow CaO + N_2}$$
  • Balance the calcium:$$\mathrm{Ca_3N_2 + O_2 \rightarrow 3\ CaO + N_2}$$
  • Balance the oxygen:$$\mathrm{Ca_3N_2 + 3\ O_2 \rightarrow 6\ CaO + N_2}$$
  • Re-balance the calcium:$$\mathrm{2\ Ca_3N_2 + 3\ O_2 \rightarrow 6\ CaO + N_2}$$
  • Balance the nitrogen:$$\mathrm{2\ Ca_3N_2 + 3\ O_2 \rightarrow 6\ CaO + 2\ N_2}$$

Double Replacement Reactions

A double replacement reaction is when two compounds react and "trade partners": $$\mathrm{FeCl_3 + Ag_2S \rightarrow Fe_2S_3 + AgCl}$$

  • The cations (\(\mathrm{Ag^+}\) and \(\mathrm{Fe^{3+}}\)) replace each other.
  • Balance the reaction:

$$\mathrm{2\ FeCl_3 + Ag_2S \rightarrow Fe_2S_3 + AgCl}$$

$$\mathrm{2\ FeCl_3 + Ag_2S \rightarrow Fe_2S_3 + 6\ AgCl}$$

$$\mathrm{2\ FeCl_3 + 3\ Ag_2S \rightarrow Fe_2S_3 + 6\ AgCl}$$
