Review

    1. What is a scatter plot and when do we use it?
      2. A graph that shows how the manipulated and responding variables are related when both are numerical measurements.
    2. Why must scatter plots be drawn and graph paper, and why is it best to use as much of the available space as possible?
      3. Graph paper helps make the graph more precise. Larger graphs can be read more precisely too.
    1. Which variable goes on which axis for a scatter plot?
      2. Manipulated variable goes on the horizontal (bottom) axis and responding goes on the vertical (side).
    2. Summarize the key features in properly labelling each axis.
      • Include the name, symbol, and units for the variable.
      • Number the squares at a fixed interval (e.g. every second square).
      • Numbering of squares must increase by a fixed amount (e.g. 0, 5, 10, 15, 20...)
      • If you do not start numbering at 0, start at a number that is a multiple of what you are counting by.
    1. State two reasons for adding a best-fit line to a scatter plot?
      2. To illustrate the trend of the data, make predications about the data, and see how scattered the data is.
    2. When would you not want to add a best-fit line?
      3. When the data has an obvious curve.
    3. Describe how to draw an appropriate best-fit line.
      4. Place your ruler on the mean (average) of both variables and rotate the ruler so that the average distance to the data points is as small as possible.
    4. Define interpolation and extrapolation.
      5. Interpolation means using the best-fit line to make predictions within the range of data. Extrapolation is when we make predictions that go beyond the range of the data.
    1. Summarize how to calculate the slope of a best-fit line.
      2. Identify two points from opposite ends of the best-fit line, and record the coordinates of both lines. Calculate the rise (difference in \(y\)-values) and run (difference in \(x\)-values) and divide to get the slope: $$k = \mathrm{\frac{Rise}{Run}} = \frac{y_2-y_1}{x_2-x_1}$$
    2. How do you determine the units of the slope?
      3. The units of the \(y\)-axis divided by the units of the(x\)-axis.
    3. What is the significance of the slope? In other words, what does the slope tell us about the data?
      4. Slope tells the ratio of how the variables change; how much the responding variable changes for each unit change in the manipulated variable.

    1. Based on how the data is presented, which variable is being treated as the manipulated variable and which is being treated as the responding variable?
      2. The manipulated variable is the first column, Circumference. The responding variable is the second column, Height.
    2. Why is it important that the data is collected only from trees of the same species and only from trees growing in the same area?
      3. These are controlled variables. If the variables are not properly controlled, we cannot make any conclusions or predictions from our data!
  2. Draw a scatter plot of the data on graph paper. Add a best-fit line.
    1. Identify the visible endpoints of your best-fit line.
      2. (0.0 m, 6.8 m) and (12.0 m, 24.8 m). Answers may vary!
