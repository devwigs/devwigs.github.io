$$\vec{\mathbf a} = \frac{\vec{\mathbf v}_f - \vec{\mathbf v}_i}{\Delta t}$$

$$\vec{\mathbf a}\color{red}{\Delta t} = \frac{\vec{\mathbf v}_f - \vec{\mathbf v}_i}{\Delta t} \color{red}{\Delta t}$$

$$\require{cancel}\vec{\mathbf a}\color{red}{\Delta t} = \frac{\vec{\mathbf v}_f - \vec{\mathbf v}_i}{\cancel{\Delta t}} \cancel{\color{red}{\Delta t}}$$

$$\vec{\mathbf a}\Delta t = \vec{\mathbf v}_f - \vec{\mathbf v}_i$$

$$\frac{\vec{\mathbf a}\Delta t}{\color{red}{\vec{\mathbf a}}} = \frac{\vec{\mathbf v}_f - \vec{\mathbf v}_i}{\color{red}{\vec{\mathbf a}}}$$

$$\frac{\cancel{\vec{\mathbf a}}\Delta t}{\cancel{\color{red}{\vec{\mathbf a}}}} = \frac{\vec{\mathbf v}_f - \vec{\mathbf v}_i}{\color{red}{\vec{\mathbf a}}}$$

$$\Delta t = \frac{\vec{\mathbf v}_f - \vec{\mathbf v}_i}{\vec{\mathbf a}}$$

$$\Delta t = \mathrm{\frac{-12.0\ m/s - 0.00\ m/s}{-9.81\ m/s^2}}$$

$$\Delta t = \mathrm{1.22\ s}$$

Algebra

  • We can perform any operation on an equation, as long as we do the same to both sides.
  • Consecutive, inverse operations cancel.
  • We can often isolate a variable by performing inverse operations in reverse order.
