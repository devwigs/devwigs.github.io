ccLatex

ccLatex is a very simple Javascript for rendering and unrendering LaTeX equations in HTML documents using codecogs.com.

Here is a display equation:

C = 2\pi r

Here is an inline equation: A = \pi r^2.

The HTML that produces the example above is:

<p>Here is a display equation:</p>
<p class="LaTeX" data-zoom="2" data-color="red">C = 2\pi r</p>
<p>Here is an inline equation:
  <span class="LaTeX" data-color="{0,0,255}">A = \pi r^2</span>.
</p>

The LaTeX markup (displayed above in red) is wrapped in an element node. We have used a <p> tag for display equations and <span> for inline equations. The tags have class="LaTeX" for easy identification.

The tags can also include optional attributes data-zoom and data-color. The data-zoom attribute specifies the desired size of the rendered equation relative to the surrounding text size. The data-color attribute is a color name recognized by LaTeX, or a decimal RGB color enclosed in curly brackets.

To render the LaTeX as PNG images, we need to load jQuery and ccLatex into our document, and then run the LaTeX function after the document is completely loaded...

<script type="text/javascript"
    src="https://code.jquery.com/jquery-2.2.3.min.js"> </script>
<script type="text/javascript"
    src="https://cdn.rawgit.com/dmaccarthy/ccLatex/v0.1.0/ccLatex.js"> </script>
<script type="text/javascript">

    $(LaTeX);

</script>

The LaTeX function can be called with two arguments. The first argument is an element or sequence of elements containing the LaTeX to be rendered. If this argument is omitted, the function will use jQuery to find all elements with a "LaTeX" class.

The second argument can be set to true to indicate that already-rendered equations should be rendered again. This is useful if the equation’s color or size has changed. If the second argument is false or omitted, equations that have already been rendered will not be rendered again.

We can remove the equation’s image and replace it with the original LaTeX markup by calling the LaTeX.unrender function. This function also accepts an element or sequence of elements as its argument, and if omitted will default to all elements with a "LaTeX" class.
